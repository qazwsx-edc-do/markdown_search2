# 《七音略校释》绪论（续）  

## 赵克刚  

### 二、四等重轻通论  

在等韵图上方，先分为唇舌牙齿喉半舌半齿七音，即七个发音部位，是横列的。各等韵图排列七音，不是一个样子，未采取位置固定法。但是，一个发音部位搬家，在其下的其他排列顺序也就依次搬过去，后者的排列顺序，始终不能离开那个发音部位。不难看出，等韵学家自发音部位而下的排列顺序，又正是采取位置固定法。在七音里，以声母的四种发音方法为横排列顺序，古人称做四等清·浊，即沈括《梦溪笔谈》所说：“每声复有四等，谓清、次清、浊、平也。”《韵会》说：“以三十六字母定每音清浊·之等。”亦称清浊为等。元刻本《切韵指掌图·检例》说：“每音有四等；全清、次清、全浊、不清不浊，如颠天田年是也。”名不同而同谓四等清浊无纷歧。四等清浊的位置在等韵图中是固定不能移的。自四等清浊位置直下，以有无介音与声母相结合，又定为四等重轻。即沈括《梦溪笔谈》所谓：“字有重、中重，轻、中轻。”必须指出：沈氏只举了名称，不是四等重轻的排列顺序，不如《七音略》·所标重轻系统完整。元刻本《切韵指掌图·检例》说“字有四等轻重，如高交骄晓是也。”以视《七音略》重轻系统，亦嫌简略。四等重轻的位置在等韵图中也是固定不能移的。以上两种固定位置，横在扩建行位，以便排列七音的清浊四等各有重轻四等；直在表示“三十六声母清浊轻重不失其伦”，即每行都上中下通同一气，不离开声母分等。不可这样：只把四等重轻孤立看待专称等。如果这样，必导致把等误会为属韵不属声母，不复承认四等清浊能称等。陈澧说：“等之云者，当主乎韵，不当主乎声。”正是这种观点。但是这种观点，在宋元等学说中找不到根据。一般人说，反切上字是定声母的，又对又不对。一个声母在等韵图上有几个位置，是定哪个位置上的声母呢？必如张麟之《韵镜·序例·归字例》所说“反切上字是同音之定位”才全对，即是说：反切上字是定声母等位的。因为，张麟之举“芳弓”切“丰”为例，以属唇音次清第三位的“芳”字为同音之定位，可以清楚看出张麟之所谓同音，是指声母同发音部位、同清浊的等位和同重轻的等位，它们是三位一体的，在等韵图直行上有它们三位一体的固定位置，而定这种行位，即定声母的等。张麟之未言等，·但定位实质为定声母的等无疑问。张麟之把定等称为定位，不以等属韵而属声母，是后人研究等、认识等最重要的理论指针。  

在汶语音韵史里，守温是发明三十六声母的人，且是以李舟 $\ll$ 切韵 $\gg$ 标韵目作四等重轻例示僧众传习而兼讲归字门法的人，可见三十六声母和等韵图从发生到发展，它们始终和韵书分不开。大致佛教徒在梵汉对音研究方面，至唐宋间，由于利用了 $\ll$ 切韵》系韵书反切系统作为汉语对音材料，要比前人学梵文所用对音汉字成体系而且字数多得多，于是就对比研究出了一个汉语声母系统，毫为三十六声母，作成了有《切韵》系韵书标韵目的三十六声母分行定位等韵图。在守温四等重轻例后，同体制的，有以《广韵》标韵目的《韵镜》以及本自胡僧《七音韵鉴》的《七音略》。设计三十六声母分行定位这种等韵图，不知佛教徒想方设法多少回，安排调整多少次，才“立韵得经纬之全”，作出七音横排四等清浊，自四等清浊而下再直排四等重轻的这种等韵图固定格式来。但是，路是走出来的。可以设想：一开始恰好是东韵，是个可以定为三十六声母清浊、重轻四等俱全的客观存在，如果表示等的声母分行定位的形式图在东韵里设计、安排、调整成功了，路也就走出来了。比如，再试东韵的上去入三个韵部，发现了可以用同样的声母分行定位形式图安排下来，·又试其他几个四声相承的韵部，或任意抽查一些韵部，更发现，无不可以用这同样的声母分行定位形式图安排下来。实践出真知，普遍对应规律豁然在眼前，它促使人如法炮制，最后就作成了《切韵》音系全部等韵图。于是可以理解：一个固定的表示三十六声母清浊重轻的等韵图，为什么各韵部都熊适用呢？各自部中来，又回到各韵部中去，它不能不这样，因而在等韵图中，韵不固定，必须变置。变置各韵部到等韵图中去，即运用等韵图到各韵部中去，与上所说，不是两种关系。不管变置什么韵部，只有一个相同的结果，即韵中的字都能为固定在等韵图中的声母清浊重轻等位所类聚。沈括《梦溪笔谈》说：“切韵之法，先类其字，各归其母。”真是通人讲通法！．不要以为等在韵。高本汉从陈澧说，等属韵，：都是看错了等韵图的荒唐话。，在等图中韵所代表的，是主要元音或主要元音兼韵尾部分，与等·无关。张麟之《韵镜·序例·归字例》说：“反切上字是同韵之对映。”“在图中逐韵属单行字母者，上下连续二位，只同一音。”即只以反切下字为同韵对映音，不论等。单行为等韵图一、三等声母行；其上下行为等韵图二，四等声母行。·在这里概括为只同一音的，：是指一等声母字可用二等声母字作反切下字，及二、四等声母字普遍用三等声母字作反切下字这些情况。也就是说：各韵一、二、三、四等声母，不论用哪个同韵字作反切下字都可以，它们与等无关，只有“同韵之对映”这种关系。反切下字不论等，只在等韵图中充当同韵对映音，必等不在韵。亦必在等韵图中韵所代表的，只是主要元音或主要元音兼韵尾部分，与韵部的韵意义相同。否则，一、二、三、四等字作反切下字，怎样能看为“只同一音”呢？等不在韵在哪里呢？在声母，张麟之早已讲得清清楚楚，在这里也表现得清清楚楚。等在声是等韵图的纲，丢掉这个纲，到与等无关的韵里寻找古人所定的等，不合，就说古人错了，真二、四等变成了假二、四等，有分别的三、四等声母变成了三、四等重纽，甚至说四等当废，等等，把古人的等韵学运用等属韵的观点研究得遍体鳞伤，本无问题而越讲问题越多,本不复杂而越讲越复杂,本不难懂而越讲越难懂，有何神益？不如返本还原，回头是岸。大致是误以属声母的四等重轻即为韵之四等吧，、不可不提提《切韵指掌图·辨分韵等第歌》：：．：：：  

见溪群疑四等连，端透定泥居两边，知徽澄娘中心纳，帮滂四等亦俱全。更有非敷三等数，中间照审义幽元。精清两头为真的，·影晓双飞亦四全。来居四等都收后，日应三上是根源。  

它说的正是四等重轻。可只在三十六声母里论四等重轻之所居、全不全，是以四等重轻属韵吗？不是。在歌句里，可以看出，.又是以七音四等清浊的排列顺序和四等重轻的排列顺序贯通一气来分辨声母等第的，是被认为不可分割的。  

，古人有无韵母概念呢？韵母，是不区别声调的，在按四声分韵部的《切韵》音系里它不可能有，不可能在 $\ll$ 切韵》音系里有韵母概念，也就不可能在等韵图里有所谓等由韵母定的事实。四等重轻，古人不离开声母分析，而韵母又是声母以外的语音部分，若谓四等重轻为不同韵母的分析排列，即所谓等由韵母定，那么二者的古今概念即不相当。.古人无韵母概念而分韵部，观点很清楚。古人分韵部，声调、主要元音和韵尾，必求相同，而主要元音以前的介音和声母，则不求相同。可以由此看出一个原则来：古人分韵所重者，在必求相同部分，下此即不再分成韵部，守界甚严，所不重者，在不求相同部分。至等韵学家，实反其道而行，他们不再重复分韵部的工作，为了利用反切土字从梵汉对音中研究汉语声母系统及汉语声母与介音相结合的特殊关系，所重者，正分韵部不求相同部分。在不求相同部分，声母只代表发音部位、发音方法，不与介音、元音相结合，大多数不易被人听见，不便口耳授受，不利于审辨声母的等韵学。于是佛教徒拿出了主意：一是学梵文辅音字母单用时带a元音的样，·用了声母带读元音来加强响度作为等韵学中审辨声母的方法，另一是看到声母带读介音，一样能加强响度，也用为等韵学中审辨声母的方法.王照《官话合声字母》采取双拼字，把声母和介音合为一个字母，注音字母也把声母和元音、介音拼读起来定名称，一样是为了加强响度，使人听得见，在方法上正古今相同。.再看，在不求相同部分，第一，若无介音，必无论清声母、浊声母，都与元音直接拼读，只有这种结合关系；而在等韵图里，与元音直接拼读的声母，同重轻。第二，若有介音，因为介.音不只一个；虽声母清浊不论，但什么介音与什么声母组群相结合，什么介音不与什么声母组群相结合，又必然各有不相同的拼读关系表现出来，而在等韵图里，与不同的介音拼读的声母，不同重轻。为了加强响度，以便审辨声母，等韵学家不仅利用了上述那些拼读结合关系，也同时分别了重轻。事实很明显，佛教徒审辨声母，不是孤立在韵书之外传音义，而是始终不离韵书进行探讨的，于是不论清声母、浊声母，它们可能有的重轻关系，即四等清浊各有四等重轻的这种关系，也就从韵书中通通概括归纳了出来。  

在《七音略》里的重轻标目，分为重轻两大类各四等.重统开口四等，即所标“重中重"、“重中轻”，而”“重中轻”无二等在其中，轻统合口四等，·即所标“轻中重”、“轻中轻，而、“轻中轻”无二等在其中。《七音略》系统完整，理论周密，在等韵学中实出于其类，拔乎其萃！罗常培先生《释重轻》谓：“重轻者，固与开合异名而同实也。”发前人之所未发，一言凿破混沌。又谓“中重中轻之别，苦难直言”，亦令人深思。一再考案《七音略》，有凡个大例被发现。.其一，不管是“重中重”也好，“轻中重”也好，只要包含有中重二字在标目中，必有照二系字（或偶而兼有其他声母三等字）·排在韵图中。可见中重在凭照二系标目，“重中重”当分开读为重、中重，“轻中重”当分开读为轻、中重。二等介音为r，是罗常培李方桂两先生提出来的。二等居四等之中，r为重浊音，古人称二等为中重，正名实两同。其二，不管是·“重中轻”也好，“轻中轻”也好，只要包含有中轻二字在标目中，必无二等字在韵图中，但是，又必有三等字在韵图中。可见中轻在凭三等标目，“重中轻”当分开读为重、中轻，“轻中轻”当分开读为轻、中轻。它们为什么都越过中重直标中轻呢？即·表示无二等字在韵图中。三等亦居四等之中，介音为i，短弱音，古人称三等为中轻，亦正名实两同。二三等名称既考定，四等呢？在等韵图里，一四等同声母，于是可以知道，在开口四等里，一等称为重，四等也就同称为重，在合四等里，一等称为轻，四等也就同称为轻，这是毫无疑义的。其三，每图只标前两等，亦以代表后两等，要按图依次寻看。如 $\pmb{\ll}$ 广韵》东韵，一二三四等俱全，《七音略》只标为“重中重”，即只标为重、中重两等，它实际代表了后两等，应依次读为重、中重、中轻、重四等。于是可以类推，在 $\pmb{\ll}$ 七音略》标为“重中轻”的，即应读为重、、中轻、重四等，无二等。以上“重中重"、“重中轻”本同为开口一类，以有无二等而分为两种排列形式，并只以前两-等标目。同样，在 $\pmb{\ll}$ 七音略》标为“轻中重”的，即应读为轻、中重、中轻、轻四等，在 $\pmb{\ll}$ 七音略 ${\gg}$ 标为“轻中轻”的，即应读为轻、、中轻、轻四等，无二等。以上“轻中重”、“轻中轻”本同为合口一类，以有无二等而分为两种排列形式，并只以前两等标目，以前只知道重表开口，轻表合口，而中重、中轻难言。由上，可见 $\ll$ 七音略》的重轻；本在分等，等与开合有区别，又自然地联系在一起。结合等看，可以进一步知道：中重在开合口俱为二等的名称，中轻在开合口俱为三等的名称，以开口而论,尚不只重表开口，中重、中轻也都能有开口呼；以合口而论，尚不只轻表合口，中重、中轻也都能有合口呼。这样，也许对四等重轻和开合的关系，得到了应有的、不失 $\ll$ 七音略 $\gg$ 原意的解释。  

再补充谈谈开合口四等，然后解释其他一些问题。在开口里，一等称为重，声母与大多数元音直接相结合。二等称为中重，声母与介音r相结合。r在中古舌齿音照二系、知二系里很稳定，即今捲舌音的来源。 $\pmb{r}$ 也可以拼读少数喉牙唇音字，可是后来变化很大：一是容易变为，如二等家 $\bf k r\3\to\bf k$ $\rightarrow$ t6ja；一是容易脱落，如二等马 $\mathtt{m r a}\to\mathtt{m a}$ 。三等称为中轻，声母与介音j相结合，j使前面的声母颚化。四等复称为重，声母与不单独使用的元音性i相结合。设声母为牙音见母 $\mathbf{k}$ ，元音为8，以上开口四等形式为：

#### （二） 轻ku·中重kru中轻kju·轻kiu  

江永《音学辨微·辨等列》说：“一等洪大，二等次犬，三四皆细，而四尤细，学者未易辨也。辨等之法，须于字母辨之。”与（一）式结合起莱可以看得很明白：一等洪大即《七音略》的第一等重，二等次大即《七音略》的第二等中重；三等细即《七音略》的第三等中轻，四等尤细即《七音略》的第四等重。江氏用洪细代替了《七音略》的“重中重”“重中轻”之类表等法，要比《七音略》简单明白容易懂得多。在江氏《四声切韵表》初稿本里，本标有“重中重”“轻中轻”之类名称，但在“三四易稿”的后来《四声切韵表》定本里他通通删去了。为什么？以审音功夫深著称的江永,自信其书胜过了《七音略》，已经化繁为简，化难为易，“开卷了然”，解决了一个“学者未易辨"等的大难题，正如他所说：“学者熟玩，音学可造精微。”四等重轻，本是为了加强响度，以便审辨声母而按声母与元音拼音、声母与介音拼音两种关系定出来的。要谈声母的等，不用重轻而用洪细也一样。一个要点是：不可-分开声母与元音、介音的配合关系（如ka，kr，kj，‘ki）单独看，必须把它们这些关系拉拢来看，而拉拢来的结果又不是落户在元音一边，而是安家在声母、声母与介音的结合体一边，即不管是重四等也好，轻四等也好，等都是定在声母上的。江氏的洪细，即跟上述重轻理论相一致，等都定在声母上。不然，怎么会提出四等洪细来告诉人们说“辨等之法，须于字母辨之”呢？四等洪细中的声母关系即重轻各四等中的声母关系。江氏所说的声母，决不是一般人所说的声母，否则，在声母上怎能辨洪细呢？必须从声母拼何元音、拼何介音看，才能说有洪音细音的不同，江氏所谓洪细，也就正是这样的。不要以为洪细指韵母，如果这样，必与江氏原意不相合，是得不出江氏“辨等之法须于字母辨之”这个结论来的，在江氏那个时代，韵母这个词还没有产生呢！可以用洪细观点结合等韵图的形式、内容总括几句：在等韵图里，一四等是声母与元音拼读的类型，以元音不同洪细而分为一四等，它们能同名称（重或轻），是从元音离开声母而声母就相同来看问题的；二三等是声母与介音拼读的类型，以介音不同洪细而分为二三等，它们不能同名称(中重、中轻)，是从元音离开这些声母介音结合体而这些声母介音结合体必不相同来看问题的。介音与前面的声母连起来，如kr、kj，可以作为复声母看，也许这样看能更好地理解古人以介音属声母的问题。·  

以r为二等介音，在解释二等音值以外,还在于区别一二等。主要是认为：一二等的区别，不在元音，在于一等无介音，二等有介音r，即认为一二等元音相同，否定了高本汉二等韵的元音与一等韵不同的说法。在等韵图里，介音属声母，定等在声母方面，韵的概念，只代表主要元音以下部分，只有张麟之所谓“同韵之对映”的关系，不是定等对象，所以，一-二三四等都能同韵部同等韵图，如东韵。在东韵一二三四等同元音，是无可否认的事实，其二等，是由照二系声母决定的，从音值看，应有介音r，再从有无介音看，一二等就很自然地区别开来了，有什么二等韵的元音与一等韵不同的地方呢？等韵学家不是按元音来分等，不能认为一二等不同元音，事实上是一二三四等都同元音的，不仅同部同韵图的东韵如此，事实上异部同韵图的其他等韵图从元音看也都如此，等韵图的观点、作法是统一的。例如 $\ll$ 七音略第二十五图·豪肴宵萧》：一等高kau，二等交 $\mathbf{k}\tau\mathbf{\bar{s}}u\rightarrow\mathbf{k}$ jar $\mathbf{1}\rightarrow\mathrm{t}6\mathbf{j}\overline{{\mathbf{a}}}\mathbf{u}$ ，三等骄 $\mathbf{\hat{x}}\mathbf{i}6\mathbf{\bar{a}}\mathbf{u}\to\mathbf{t}\mathbf{j}\mathbf{\bar{a}}\mathbf{u},$ ，四等浇 $\mathbf{kiau}{\rightarrow}\mathbf{t6jau}$ 。又如 $\ll$ 七音
