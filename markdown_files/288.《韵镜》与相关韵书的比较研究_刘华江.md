# 摘要  

《韵镜》是我国早期的一种根据韵书，主要是韵书中的小韵首字制字的韵图，也是研究中古汉语乃至整个汉语史的重要资料。但由于《韵镜》一书特殊的流传过程，其作者现已不可考。众多前贤对《韵镜》一书进行了不少的研究，解决了很多重要的问题，从而使对《韵镜》一书的研究不断得以深入。但关于《韵镜》一书的成书年代及其是根据何种韵书而作，一直到目前学术界仍未达成一致的意见。问题的本质在于，到目前为止还没有人对《韵镜》一书与相关各韵书之间的关系进行全面、详细、系统的比较研究，因而也就难以得出比较确定的能为大家都普遍接受的结论。本文即采用比较的方法，通过《韵镜》与相关各韵书之间的比较研究，全面考察《韵镜》与相关韵书之间的关系，从而得出有助于上述问题解决的办法和途径。本文除引论和结论部分外，第一章和第二章展开了《韵镜》与相关各韵书之间的全面比较。第一章比较了《韵镜》与相关各韵书的韵目及其次序，从一个方面确定了《韵镜》列字所据为何种韵书的可能性和范围。第二章比较了《韵镜》所列之字与相关各韵书所收之字，主要是小韵首字之间的关系，从另一个方面确定了《韵镜》据何种韵书列字的最大可能性。通过全面的比较和直观鲜明的统计数据，我们认为《韵镜》当是据《唐韵》与李舟《切韵》之间的某种韵书而作。这种比较不仅考虑了《韵镜》与相关各韵书在韵目及其排列次序上的特点，更重要的是还考虑了《韵镜》与相关各韵书列字收字的情况，具有更高的科学性。

# 引论  

# 第一节 关于《韵镜》一书的作者及其成书年代  

《韵镜》是我国现存最早的等韵图，也是研究中古汉语乃至汉语史的重要资料之一。然而因其特殊的产生过程和流传过程，《韵镜》的作者现在已不可考。关于该书的成书年代，也仅仅能得其模糊梗概。早在南宋初张麟之刊行《韵镜》之时，张氏已有疑问，张氏在其序中说：“自是研究，今五十载，竟莫知源於谁。”自从张氏以来，已有不少学者对《韵镜》一书的成书年代作出了多方面的考求，但结果到目前为止依然是莫衷一是，没有定论。关于《韵镜》一书的成书年代，目前主要有以下几种看法。  

# 一、主张唐元和时神珙所作  

宋张麟之在其於嘉泰三年重刊《韵镜》所作的序中说：“《韵镜》之作，其妙矣夫！余年二十始得此学字音。往昔相传，类曰洪韵，释子之所撰也。有沙门神珙（恭拱二音），号知音韵，尝著切韵图载玉篇卷末，窃意是书作於此僧，世俗讹呼珙为洪而，然又无所据。”张麟之怀疑《韵镜》乃为神珙所作，但张氏又没有确实的证据，可见张氏所说全是揣测之辞。正如赵荫棠在《等韵源流》一书中对此所作的批评那样：“张氏把《韵镜》归於神珙，不见得确实。因为《洪韵》，未必是《珙韵》之误。朱子语录卷百四十第十二页有一条云：‘洪州有一部洪韵，太平州亦有部韵家文字。’依此而言，安知‘洪’非指地域而言？”「由此可见，张氏的说法不足为据。  

# 二、主张原型出於隋唐  

日本学者大矢透即主张此说。大矢透在其所作的《韵镜考》一书中主张《韵镜》一书成於隋末唐初。据台湾学者孔仲温在其《韵镜研究》一书中所述，大矢透氏的证据有二：  

〈一）日释安然悉昙藏引武玄之韵诠明义例，所述之正纽相证，傍通取韵及阙位等等情形，俱与《韵镜》合符。  

（二）日人藤原佐世之日本现在书目载切韵图一卷。  

赵荫棠以为此说仍有未当处，赵荫棠在《等韵源流》一书中以为韵诠尚是一部未曾脱离反切范围的韵书，而日人藤原佐世所谓的切韵图又安知非如张麟之以玉篇卷末之四声五音九弄反纽图为切韵图。²由此可见，大矢透之说亦有未能令人信服之处。  

# 三、主张不创自宋人  

罗常培并不完全同意大矢透氏的主张，罗先生在《通志七音略研究》一文中主张《韵镜》当是效法悉昙章之韵图，并认为《韵镜》是在“切韵成书以后产生，而非创自宋人。”罗先生的证据主要有三：  

〈一）张麟之在《韵镜序作》题下注云：“旧以翼祖讳敬，故为韵，今迁挑庙，复从本名。”翼祖乃宋太祖追封其祖之尊号，如《韵镜》作于宋人，则宜自始避讳，何须复从本名？如果《韵镜》有旧名，则《韵镜》必出于前代。此为《韵镜》不创自宋人之一证。  

〈二〉以《韵镜》之韵目而言，其置覃、咸、盐、添、谈、衔、严、凡八韵 於侵韵之后，实与唐李舟切韵，宋陈彭年广韵等为一系，而《韵镜》之转次，则 显然依李舟一系重加排定，与据陆法言切韵、唐孙唐韵一系排定的《七音略》 略异，然自其殿以蒸、登者，犹可窥见原型与《七音略》同源。此为《韵镜》不 创自宋人之二证。  

《三）据敦煌唐写本守温韵学残卷所载“四等重轻例”之分等，与《韵镜》悉合，所以四等之分劃，在守温以前蓋已流行，而当起源於唐代，殆无可疑。此为《韵镜》不创自宋人之三证。  

罗先生的主张，认为《韵镜》一书非创自宋人，即《韵镜》一书当创於宋代以前，其说有理有据，很有说服力，后来的学者也多同於此说。  

# 四、主张起源於隋唐尚在疑似之间  

此是赵荫棠的主张。赵先生不完全同意罗常培的意见，因而赵先生对《韵镜》一书的撰述年代持一种怀疑态度，从而有“所以我以为《韵镜》起於隋唐的话，尚在疑似之间”的话。3  

# 五、主张成书於张麟之初次刊行或陈彭年重修广韵前  

这是王力在《汉语音韵学》中所持的态度。王力就张麟之於宋绍兴辛已刊行时所作的序，从而认为《韵镜》至少是宋绍兴辛已以前的作品，但王力同时又将《韵镜》一书的成书年代置疑於陈彭年重修《广韵》以前。王先生的态度比较谨慎，主张因此显得有些保守。  

# 六、主张成书於宋  

李新魁即主张此说。李新魁在其《韵镜研究》一文中提出：“韵图作者在编制韵图之时，一般也都采用所据韵书的小韵首字，因为这样做最省事、最便当。我们很难设想有这样的作者，他放着韵书的小韵首字不用，而偏去找小韵中其他的字，除非这种完全出于音理上的必需，不然就是这位作者故意自找麻烦。因此，使用韵书中的小韵首字成为韵图列字的通则。”“《韵镜》是什么时代的作品？它是依据什么韵书而作？要弄清楚这些问题，最好的办法是拿《韵镜》的列字与各种韵书作全面的比较。”李先生在这篇文章中得出的结论是：“我们通过对《韵镜》的全面研究，认为它应作于宋代。”（李新魁，《《韵镜）研究》，《李新魁语言学论集》，中华书局，1994）李先生在这篇文章中提出的研究思路和方法具有方法论的意义，但由于李先生并没有如自己所说的那样对《韵镜》作全面的比较，而只是将《韵镜》与《广韵》、《集韵》及《礼部韵略》作比较，因此所得出的结论当然是不能令人信服的。具体情况请参照李先生的《韵镜研究》一文。  

七、主张成书於宋，底本则据宋以前  

此是董同的主张。董同在其所作的《汉语音韵学》一书中主张：“今传最早的等韵图有郑樵《通志》中的《七音略》与南宋绍兴年间张麟之刊行的《韵镜》。这两种韵图虽然都出于宋代，不过他们都前有所据。他们的底本，从下列三事可以看出，都是宋以前的作品。”董先生举出了三个方面的证据：.  

（1）《七音略》的底本，郑樵自己说是《七音韵》，《韵镜》的底本，张麟之说是《指微韵镜》；提到他们，张麟之又说：“..其来也远，不可得而指名其人，故郑但言梵僧传之，华僧续之。……”  

（2）张麟之《韵镜序》下有注云：“旧以翼祖讳敬，故为韵，今迁桃庙，复从本名。”所谓“翼祖”是指宋太祖的祖父，如果《韵镜》作於宋人，起始就该避讳叫《韵》，根本无所谓本名。现在本名既为《韵镜》，出於宋代之前，自  

无疑义。  

（3）《七音略》把覃谈咸衔盐添严凡诸韵列在阳唐庚耕清青之前，《韵镜》 与《七音略》又有蒸登韵列在最后。由此可见他们根据的韵书实在比《广韵》早， 而《七音略》所据更比《李舟切韵》早。  

八、主张《韵镜》一书是“存累造成的”  

此是鲁国尧在《卢宗迈切韵法述评》一文中提出的比较新的学说。（鲁国尧，《（卢宗迈切韵法〉述评》，中国语文，1992；《〈卢宗迈切韵法）述评》续；中国语文，1993）吾师杨军在其所著的《韵镜校笺》一书和《〈韵镜〉所标“开合”及相关问题再研究》、《佛经翻译与切韵图的产生和发展》等文章中又对此新说作出了比较详备的阐述。此一学说现在已经逐渐得到学术界的认同。同时杨军也对此提出了自己的看法，“我们没有确切的证据质言切韵图最早出现的具体时间，但根据它们产生的条件来推断，或许就在隋末唐初，至迟不应晚於八世纪。”“罗先生认为中国早期切韵图是唐代僧人仿悉昙体制，总摄《切韵》音系的产物，‘自《切韵》成书后即当继之以生’。我认为罗先生的看法有理有据，可以接受。而《韵镜》一书的最后定型，也应该在唐代。因为我们发现，制作《韵镜》所依据的是唐代的某一种韵书。”“总结以上，我们可以得出下面的结论：大约在唐代初期，高僧中有人为了译音的准确和便利，在‘悉昙’的启发下，编制了一种这种符合汉语语音特点的音节结构表。这种音节表就是早期的切韵图，也就是《韵镜》、《七音略》这类韵图的雏形。”（《佛经翻译与切韵图的产生和发展》，第五届中古汉语国际学术讨论会论文，杨军，2005；一见《韵镜校笺》，杨军，2007)  

总之，关于《韵镜》一书的成书年代，主要有以上几种看法。但《韵镜》一书到底产生於什么年代，至今仍然没有统一的意见。李新魁曾说过：“《韵镜》是什么时代的作品？它是依据什么韵书而作？要弄清楚这些问题，最好的办法是拿《韵镜》的列字与各种韵书作全面的比较。”（李新魁，《〈韵镜》研究》，《李新魁语言学论集》，中华书局，1994）这种比较是可行的，也是很有意义的。目前学术界还没有人对此进行过全面的比较。因此本文就将《韵镜》与相关的一些韵书作一个全面的比较，试图从中找出一些有助於此一问题解决之线索和证据。  

# 第二节 关于《韵镜》一书的流传过程及其层累的性质  

# 0.2.1关于《韵镜》一书的流传过程  

《韵镜》的流传过程非常特殊。正如台湾学者孔仲温在其著作《韵镜研究》中所说：“韵镜之刊行，自张麟之南宋高宗绍兴三一年（西元一一六一年）序观之，此时为初次刊行，其后经历三十六年，至宋宁宗庆元丁已（即庆元三年，西元一一九七年）再度发行，今之永禄本韵镜於绍兴辛已序后有‘庆元丁已重刊”木记，由此可知也。又经六年至宁宗嘉泰三年（西元一二〇三年)，张麟之复撰序文，依此序又可推断，此时乃张麟之第三次刊行韵镜。张氏由高宗绍兴三一年到宁宗嘉泰三年这四十二年中，总计刊行三次，可见当时韵镜之流传，尚称普遍。”然而非常奇怪的是，《韵镜》自张麟之三刻以后，竟然煙灭不传，在我国的古代史志和相关文献里，也找不到有关这部书的著录。更为奇怪的是，虽然《韵镜》一书在我国本土没有流传与著录，但却流传於彼邦日本。据台湾学者孔仲温在其著作《韵镜研究》一书中的研究，《韵镜》一书传入日本的时间是，“由其中署有‘建长四年二月十二日书写了明了房信之题记，可由明了房信抄写韵镜之时间，而推断流传日本最晚年限即是此年。建长四年相当吾国南宋理宗淳祐十二年(西元一二五二年)，故於理宗淳祐十二年以前，此书已在日本；至於传抵彼邦之上限，高师仲华又指出自明了房写本后，至嘉吉元年写本，享禄元年刊本之书,皆载有张麟之宋宁宗时嘉泰三年（西元一二0三年）序，故韵镜流传日本理应在嘉泰三年以后，就此二端，高师仲华为之推论云：  

由此二证，可知韵镜之传入日本，必在宋宁宗嘉泰三年（即西元一二 $\pmb{\theta}$ 三年）与宋理宗淳祐十二年（即西元一二五二年）之间，其为期约五十年，惟不知为五十年中之何年耳。故今言传入之时间，仅得五十年之范围而已，若欲确指为某年，则非目前史料所可稽考者矣！”  

《韵镜》传入日本以后，不仅在日本得以继续流传，而且拥有众多的研究者。因此《韵镜》一书具有很多种写本，而且由于抄写者不同，各本之间互有参差不同之处。这些不同之处有些是由于抄写的错误，有些是中日对于汉字的写法不同的问题，有的是抄写者、校读者或研究者对其作出的各种注释或者是研究的心得，还有很多是后人根据后来的韵书等对其的改定和添加，如此等等，纷繁复杂。直到清代末年，黎庶昌出任驻日公使，偕杨守敬访求本土失传的中国古籍，得永禄本《韵镜》一种，遂於光绪十年（1884）在日本使署刊入《古逸丛书》之十八。自此，《韵镜》一书在日本流传长达600多年后才最终得以回归故土。因此，我们有充分的理由相信，从日本流传归来的《韵镜》已远非当年流传入日本时的《韵镜》了。关于《韵镜》一书的流传情况，请参考杨军的《〈韵镜〉刊行与流传情况再研究》和《佛经翻译与切韵图的产生和发展》。  

# 0.2.2关于《韵镜》的层累性质  

由于《韵镜》一书的特殊的流传过程，造成了《韵镜》一书中出现了许多很难理解和解释的地方，学术界也因此争论不休，在一些问题上一时难以达成一致的意见。鲁国尧在《卢宗迈切韵法述评》一文中提出《韵镜》一书是经过长期的层累过程而形成的。（《〈卢宗迈切韵法）述评》，中国语文，1992；《〈卢宗迈切韵法）述评》续，中国语文，1993）杨军在其所著的《韵镜校笺》一书和《《韵镜>所标“开合”及相关问题再研究》、《佛经翻译与切韵图的产生和发展》等文章中又对此新说作出了比较详备的阐述。杨军认为：“从时间及流传过程来看，《韵镜》的层累又可以分为两个不同的层次，第一是从该书的雏形到张麟之刊刻为限；第二是该书流传到日本至回归本土为止。我在《《韵镜》所标“开合”及相关问题再研究》（2005）中说：“早期韵图的‘层累’具有两种性质，第一种是《韵镜》之类的韵图从草创到成熟，不断有人要对它进行‘完善’，在这个过程中，出现了用后起的韵书补充和改动原书列字的现象。这种性质的‘层累’从时间上可以划在张麟之首次刊刻《韵镜》以前,亦即宋高宗绍兴辛已年(绍兴三十一年，1161)以前。第二种‘层累’发生在《韵镜》流传到日本以后，时间大约是1203一1252年以后。造成的原因一方面同样是有人想对它进行‘完善’，因而把《广韵》、《集韵》里的一些字注在相应的位置，甚而有的直接增补到原书中去了。另一方面是有人为了纠正该书流传过程中产生的错讹，於是根据《广韵》或《集韵》来判定原书的是非，这种做法无论动机如何，客观上还是淹晦了原书本来的面貌。再一种情况是日本学者往往为原书所列的字加注同音字以及注音、注义等等，或为原书所列的字或标注等加注校字、校语等，后来在抄写时就出现了旁注入原书或用批注改动原文的情况。经过一次又一次的增改，我们现在看到的《韵镜》等早期韵图里出现了若干原书没有的内容，产生了好些难以解释的问题。因此，只有从理念上充分认识《韵镜》的层累性，才能对该书的各种复杂情况作出正确的判断；也只有真正认识到《韵镜》等早期韵图的层累性质，剥除掉后人累加的成分而完成对这些韵图的复原工作，.才可能对它们进行出正确合理的构拟或重建（reconstruction）。”（《佛经翻译与切韵图的产生和发展》，第五届中古汉语国际学术讨论会论文，杨军，2005；一见《韵镜校笺》，杨军，2007）由此可见，我们只有充分认识到《韵镜》一书的“层累性”，我们才能对《韵镜》中出现的一些现象作出较为合理的解释，我们对《韵镜》的研究也才能更进一步。本论文原拟写五章，其中有一章即为“剔除部分累增成分后的《韵镜》列字与相关韵书收字主要是小韵首字之比较”，但由于篇幅有限和有些问题尚未考虑成熟未能在本论文中体现，姑且留待以后再作研究。  

# 第三节 《韵镜》研究综述  

关于《韵镜》一书的研究概况，由于本人所了解的材料和篇幅有限，在这里只能就一些有代表性的，有影响力的简要地说一说。这里分两个方面来说。  

# 0.3.1《韵镜》在国外的研究概况  

《韵镜》在国外的研究主要集中在日本。众多研究《韵镜》的日本学者对《韵镜》一著的产生和形成，《韵镜》的版本和流传，《韵镜》的体例和内容，《韵镜》的语音性质等问题都有很深入地研究，也取得了很丰富的研究成果。关于《韵镜》在日本研究的具体概况可参考李无未的《日本学者的〈韵镜）研究》一文（《日本学者的〈韵镜）研究》，李无未，古汉语研究，2004)，具体情况这里不再赘述。  

# 0.3.2《韵镜》在国内的研究概况  

关于《韵镜》在国内的研究概况，这里分两个方面作个简要介绍。《韵镜》在台湾的研究概况，《韵镜》在大陆的研究概况。  

# 0.3.2.1《韵镜》在台湾的研究概况  

关于《韵镜》在台湾的研究概况在这里主要说一说孔仲温的《韵镜研究》。孔仲温在《韵镜研究》一著中对《韵镜》的源流，《韵镜》的撰述年代，《韵镜》的流传情况，《韵镜》的内容以及《韵镜》的音系都有比较详细的介绍和研究。  

此书对研究《韵镜》的后学者有很重要的参考价值。关于《韵镜》在台湾的研究概况，这里还要说一说龙宇纯的《韵镜校注》一著。该书比勘作者所见《韵镜》各本异同，并参考有关韵图韵书作为校注，共出校记838条。此书分图作校，一目了然，易于检阅和查找，对研究《韵镜》的后学者有很高的参考价值。  

# 0.3.2.2《韵镜》在大陆的研究概况  

关于《韵镜》在大陆的研究概况，这里想说说李新魁的《韵镜研究》和《韵镜校证》，陈广忠的《韵镜通释》，李无未的《日本学者的〈韵镜》研究》，杨军的《韵镜校笺》。李新魁的《韵镜研究》一文是大陆研究《韵镜》比较早的成果，该文具有很重要的方法论意义，即明确指出要弄清楚《韵镜》是什么时代的作品，《韵镜》是依据什么韵书而作，最好的办法是拿《韵镜》列字来与各种韵书作全面的比较。但该论文又具有很明显的局限性，即论文的作者只是将《韵镜》与《广韵》、《集韵》及《礼部韵略》作比较，因此得出的结论也很难让大多数人信服。关于这一点张世禄这样说过：“因为从韵书的变迁史看来，《广韵》承袭唐人而作（详拙著《广韵研究》第二章)，而《韵镜》、《七音略》二书所根据的，也是唐人韵书，并非宋代的《广韵》。所以考证《韵镜》、《七音略》的音读，与其用宋初的音或《广韵》一书为根据，不如用唐人韵书或现今所测定的隋唐音为根据。”（《张世禄语言学论文集》，学林出版社，1984）李新魁的《韵镜校证》也是国内在《韵镜》校勘方面比较早的专著，该著将《韵镜》列字与相关韵图列字和韵书收字相互比勘，共出校记1219 条，解决了《韵镜》列字上的许多问题，也给研究《韵镜》的后学者提供了经验和借鉴。在国内研究《韵镜》的专著还有陈广忠的《韵镜通释》，这是一部属于通论性质的著作，主要是对《韵镜》中涉及到的一些问题进行诠释。关于《韵镜》在大陆的研究概况，这里还要说一下李无未的《日本学者的〈韵镜》研究》一文，该文较全面详细地介绍了《韵镜》在日本的研究概况，对我们研究《韵镜》的后学者了解《韵镜》在日本的研究概况有一定的帮助。关于《韵镜》在大陆的研究概况，最后要说一说的是杨军的《韵镜校笺》一著。杨军的《韵镜校笺》一著在吸收和借鉴前人研究的成果上，将目前国内外所能见到的众多《韵镜》本子，相关的韵图以及众多韵书包括很多残卷相互比勘，无论在研究的广度上还是在研究的深度上都有很大提高，提出了不少新的见解和观点，也解决了很多前人未能解决的重要问题，可以说是代表了目前国内研究《韵  

镜》的最高水平。  

总之，经过众多前贤的不懈研究，《韵镜》研究取得了可喜的成绩，这里不再一一赘述。但总的来说，目前研究《韵镜》的力度还不够，尤其是国内研究《韵镜》的学者还太少，还有很多与《韵镜》有关的问题还没有人去进行考察与研究。本文即选取其中的一个角度，从韵图和韵书相互关系的角度来对《韵镜》进行研究。关于此一研究，很多前贤虽然已经注意到并在他们的论著中有所体现，但明确提出并进行如此全面的比较研究恐尚属首次。因此，本文的研究对于整个《韵镜》的研究无疑是有意义的。  

# 第四节 本文研究的方法、所使用的材料及研究的局限性  

0.4.1本文研究的方法  

李新魁曾经说过：“韵图作者在编制韵图之时，一般也都采用所据韵书的小韵首字，因为这样做最省事、最便当。我们很难设想有这样的作者，他放着韵书的小韵首字不用，而偏去找小韵中其他的字，除非这种完全出于音理上的必需，不然就是这位作者故意自找麻烦。因此，使用韵书中的小韵首字成为韵图列字的通则。”（李新魁，《<韵镜）研究》，《李新魁语言学论集》，中华书局，1994）杨军也认为，“既然《韵镜》是为译经选字而定，不列小韵首字何以方便检索韵书？所以必列小韵首字。”因此本文就循着李新魁和杨军的思路，采用比较的方法，将《韵镜》所列之字与相关的一些韵书所收之字，主要是小韵首字作一个全面的比较，比较分析它们之间的异同情况，从而得出有助于相关问题解决的证据。

六、《广韵》。采用周祖谟《<广韵）校本》所用的底本，即张氏重刊宋本《广韵》，泽存堂藏板。下文简称为《广韵》。  

# 0.4.3本文研究的局限性  

本文研究的局限性主要表现在两个方面。局限之一是现存韵书仅为有限的几种，在现存的这些韵书中保存完整的又是更为有限。因此本文所说的全面比较，除了保存完整的唐写本王仁刊谬补缺切韵，即“王三”和《广韵》是与《韵镜》作完整的比较以外，其他各种韵书由于都是残卷，因此所说的全面比较只能是将现存的部分全部和《韵镜》的相关部分作比较。局限之二是由于《韵镜》一书的特殊成书过程，《韵镜》在流传过程中的层累导致了一些列字上的混乱。如何识别并剔除掉这些后人所累增的成分以还《韵镜》原有的面目，再将.《韵镜》的原目与相关韵书进行比较，这无疑是一项很有意义的工作。这两个方面的局限都给我们的比较造成了一定的困难。但由于在现存的韵书中每一个阶段都有代表性的著作传世，在这些传世的韵书中，有些是完整的，即使是有残缺的，如S2071,P2011等，其所保存的部分还是有相当大的比例的，它们虽是残卷，但还是可以代表该种韵书的，因此本文的这种全面比较研究还是很具有科学性的。同时，在某种程度上我们也能识别《韵镜》在层累中所累增的一些成分，注意到这些，我们对《韵镜》的研究将会更进一步。本人根据杨军的《韵镜校笺》一著已对此进行了一定的整理，本拟作“剔除部分累增成分后的《韵镜》列字与相关韵书收字主要是小韵首字之比较”作为本文的一个组成部分，但由于篇幅有限和有些问题尚未考虑成熟未能在本文中体现，姑待以后作进一步的研究。  

# 第一章 《韵镜》与相关韵书韵目之比较  

# 第一节 《韵镜》与相关韵书韵目的比较  

这里将《韵镜》的韵目与相关韵书的韵目作比较，并作为单独的一章，一方面是因为韵目同样是《韵镜》一书和相关韵书的一个有机组成部分，另一方面是因为此一问题关系重大，对判断《韵镜》一书到底是依据何种韵书，进而对断定《韵镜》一书的成书年代都有很重要的意义。关于此一问题，前辈学者论述颇丰，对我们的启发很大，我们首先有必要回顾一下。  

罗常培在其《通志七音略研究》一文中认为《韵镜》一书是在“切韵成书以后产生，而非创自宋人”，对此罗常培列举了四个方面的证据，其中与韵目及其排列次序有关的一个方面的证据是：“《七音略》之转次，自第三十一转以下与《韵镜》不同：前者升覃、咸、盐、添、谈、衔、严、凡於阳、唐之前，后者降此八韵於侵韵之后。案隋唐韵书部次，陆法言《切韵》与孙《唐韵》为一系，李舟《切韵》与宋陈彭年《广韵》等为一系。前系覃、谈列阳、唐之前，蒸、登居盐、添之后；后系降覃、谈於侵后，升蒸、登於尤前。今《七音略》以覃、谈列阳、唐之前，实沿陆、孙旧次，特以列图方便而升盐、添、咸、衔、严、凡与覃、谈为伍。至於《韵镜》转次则显依李舟一系重加排定，惟殿以蒸、登，犹可窥见其原型本与《七音略》为同源耳。此二证也。”  

张世禄在其《中国音韵学史》（下）中也论述了相关思想，“此外王圻续文献通考又载有宋崔敦诗《韵》及宋吴恭《七音韵镜》等书，和《七音略》所据的《七音韵》及《韵镜》，在名称上也很类似；可以见得在南宋以前，因为《切韵》一派韵书的风行，这些相类似或同一系统的等韵书，也随之蜂起。大概们互相雷同的过多，在张麟之那时（南宋初年）已经觉得此等书‘其用也博，其来也远，不可得指名其人’了。（注四十三）所谓‘其用也博，其来也远’，正是因为们是用来说明韵书上的切语，所以和当时最通行的《切韵》一派的韵书相辅而行；们的原型必起于唐代，在《切韵》一类的韵书通行之后就发生的。罗常培在《通志七音略研究》一文里曾列举四证，以明等韵图为唐人所创；兹节录之如左：·.”（张世禄《中国音韵学史》下，上海书店，1984）此处张世禄是借引用罗常培在《通志七音略》研究一文中所举出的证据来表达自己与之相类似的观点的。同时张世禄在《等韵学派系统的分析》一文中又说：“等韵表上，因排列等呼的关系，不能不把韵书的目次稍加颠倒；可是他们究竟是根据哪一种韵书而作的，也可以从序次上看出来。《韵镜》和《七音略》为何都把蒸、登二韵放在最末尾的两个图内？为何《七音略》把属于覃、谈诸韵的各图和属于阳、唐韵各图放在一起？因此可以断定《韵镜》、《七音略》两书，他们原来的根据，并不是宋代的《广韵》，而是孙氏的《唐韵》。”“魏了翁《韵镜后序》说：‘今韵降覃、谈于侵后，升蒸、登于青后’，足以见得宋人韵书是如此而和《唐韵》原次不同。《七音略》三十四图以上的序次，覃、谈在歌、麻之后，是依据于孙氏《唐韵》的。又把盐、衔、严、凡升上来，乃是为了排列等呼和配置开合的关系，不得不如此。《韵镜》把侵、覃以下九韵排在三十九图以下，显然是依据魏氏所谓的‘今韵’加以改排的；可是，蒸、登二部排在最后，仍留了一个不可磨灭的痕迹。我们从这韵部次序上观察，一方面可以断定，《七音略》和《韵镜》两书同出于一种蓝本，而与其说是为《广韵》而作，不如说他们原来都是为《唐韵》而作。”（《张世禄语言学论文集》，学林出版社，1984)  

董同在《中国语音史》一书中对此也有论述。“今传最早的等韵图有郑樵通志中的《七音略》与南宋绍兴年间张麟之刊行的《韵镜》。这两种韵图虽然都出于宋代，不过他们都前有所据。他们的底本，从下列三事可以看出，都是宋以前的作品。〈1）七音略的底本，郑樵自己说是‘七音韵’，韵镜的底本，张麟之说是‘指微韵镜"；提到他们，张麟之又说：‘…··其来也远，不可得而指名其人，故郑但言梵僧传之，华僧续之。…’（2）张麟之韵镜序下有注云：‘旧以翼祖讳敬，故为韵，今迁桃庙，复从本名。’所谓‘翼祖’是指宋太祖的祖父,如果韵镜作於宋人，起始就该避讳叫‘韵’，根本无所谓本名。现在本名既为《韵镜》，出於宋代之前，自无疑义。（3）七音略把覃谈咸衔盐添严凡诸韵列在阳唐庚耕清青之前，韵镜与七音略又把蒸登韵列在最后。由此可见他们根据的韵书实在比广韵早，而七音略所据更比李舟切韵早。”（董同龢《中国语音史》，华岗出版有限公司，1978）  

日本人平山久雄在《〈韵镜》二事》一文中也说：“上述张文云：‘魏了翁《唐韵后序》说：‘今韵降覃、谈于侵后，升蒸、登于青后’，足以见得宋人韵书是如此而和《唐韵》原次不同。··《韵镜》把侵、覃以下九韵排在三十九图以下;显然是依据于魏氏所谓‘今韵’加以改排的；可是，蒸、登二部排在最后，仍留了一个不可磨灭的痕迹。”“那么，还有什么方法能说明《七音略》与《韵镜》之间转次的不同？我想，几乎惟一的办法是假定曾经有这么一种切韵系统的韵书，其麻韵以下的韵次是（逗号表示摄的界线)：麻，阳唐，庚耕清青，尤侯幽，侵，覃谈盐添咸衔严凡，蒸登”。（平山久雄《<韵镜》二事》，《纪念王力九十诞辰文集》，《纪念王力九十诞辰文集》编委会编，山东教育出版社，1991）在此文中，平山久雄不仅与前面的学者一样提出了《韵镜》的韵目是依据后人的韵书加以改排的，蒸、登二部排在最后是一个不可磨灭的痕迹，而且进一步提出了假设，即在《切韵》系韵书的发展史上，曾经出现过这么一种韵书，它的韵目和排列次序与今《韵镜》的韵目及其排列顺序是一致的。

以上我们引用并分析了有关《韵镜》韵目及其排列次序的论述，并将《韵镜》与有关韵书的韵目及其排列次序列成了一个综合比较表格，以便对其进行更直观的考察。从以上的论述和综合比较表格可以看出，在韵书的发展过程中，从韵目及其排列次序上来看有以下一些特点：  

第一、“蒸、登”、“覃、谈”的位置（举平包括上去)  

从“蒸、登”、“覃、谈”所处的位置来看，在韵书的发展过程中呈现出两个不同的阶段。第一个阶段是从S2071、P2011、“王三”一直到《唐韵》（包括“五代刻本韵书")，“蒸、登”的位置比较靠后，是放在“咸、衔、严、凡”四韵之前，不与其他收-n 尾的“阳、唐”等韵放在一起；而“覃、谈”的位置相对靠前，放在“歌、麻”之后，“阳、唐”之前，不与收 $\mathbf{\nabla}\cdot\mathbf{m}$ 尾的“侵、盐、添、咸、衔、严、凡”等韵放在一起。第二个阶段是从李舟《切韵》到《广韵》，“蒸、登”的位置相对靠前，放在收-n尾的“阳、唐、庚、耕、清、青”等韵之后；而“覃、谈”的位置也降到“侵”、“盐”之间，而与收- $\mathbf{\nabla}\cdot\mathbf{m}$ 尾的“侵、盐、添、咸、衔、严、凡”等韵放在一起。而把“蒸、登”两韵放在青韵之后，和“阳、唐、庚、耕、清、青”等韵连在一起，把“覃、谈”两韵放到侵韵之后，和“侵、盐、添、咸、衔、严、凡”等韵连在一起，从而使收-n、-)、-m 三种韵尾的韵相连有序当自李舟《切韵》始。（参见濮之珍《中国语言学史》，上海古籍出版社，1987）而《韵镜》也是将“蒸、登”排在最后的，这正与前一阶段的韵书在处理“蒸、登”韵的次序上比较接近。如果依照众多前贤所说的《韵镜》将“蒸、登”排在最后乃是体现其所依韵书之旧的话，那么《韵镜》所依据的韵书当在李舟《切韵》之前。（“五代刻本韵书”中的“蒸、登”也是靠后的，“覃、谈”也是靠前的，仅依此不能排除依据“五代刻本韵书”的可能性。)  

# 第二、四声相承之关系  

从四声相承的关系来看，也体现出韵书在发展过程中经历了两个不同的阶段。从S2071、P2011、“王三”一直到《唐韵》（包括“五代刻本韵书")，其入声的韵目在排列次序上都是相当混乱的，不与平上去三声相承，且各韵书入声韵目的前后排列次序也基本上趋向一致。而从李舟《切韵》到《广韵》，入声韵目与平上去三声相承，次序井然。入声与平上去三声一一相配也是自李舟《切韵》始。（参见濮之珍《中国语言学史》，上海古籍出版社，1987）而《韵镜》的平上去入四声也是紧密相承的。如果从四声相承的关系来看，《韵镜》所依据的韵书当在李舟《切韵》之后，或是在李舟《切韵》之前不远，但应当在孙《唐韵》之后。同时也可以据此排除《韵镜》列字依据五代刻本韵书的可能性。  

综合以上两种情况，《韵镜》列字当是依据孙《唐韵》之后，李舟《切韵》之前的切韵系的某种韵书，除非是《韵镜》的制作者对《韵镜》的韵目及其次序作过某种调整。在理论上这种韵书居于孙《唐韵》之后，李舟《切韵》之前，所以既能符合前一阶段韵书的特点，即“蒸、登”的位置比较靠后，“覃、谈”的位置相对靠前，又符合后一阶段韵书的特点，即四声相承，次序井然，这也正体现了韵书在发展过程中的一种过渡和前后相承的关系。同时，这也与平山久雄在《韵镜二事》一文中所作的推论和假设是相一致的。当然《韵镜》列字到底依据何种韵书，还有待其他方面的证据。  

# 第二章“《韵镜》列字与相关韵书收字主要是小韵首字之比较  

本章所说的《韵镜》列字与相关韵书收字主要是小韵首字之比较，指的是将《韵镜》正图位置中所收的字与相关韵书所收之字主要是小韵首字的比较。这里的比较暂不考虑《韵镜》层累过程中所累增的成分，按照现存永禄本《韵镜》正图位置中所列之字来与相关韵书所收之字主要是小韵首字作比较。  

以永禄本言之，《韵镜》一书共有二百零六韵，其中上平声二十八韵，下平声二十九韵（《韵镜》中看不出上下之分)，上声五十五韵，去声六十韵，入声三十四韵。《韵镜》一书不计韵目和标明其体例的字以外，共列字三千八百九十三个。其中上平声五百七十一个字，下平声五百八十四个字，上声九百四十四个字，去声一千零八十二个字，入声七百一十二个字。这其中还不包括以下的情况（后人注音)：内转第五合图中的“音追”；外转第十四合图中的“仕怀反”；外转第二十二合图中的“丁全反”；内转第二十八合图中的“去靴反”；内转第二十九开图中“下反”；外转第四十合图中的“白甘反”；外转第四十一合图中的“孤女法反”，“羯起法反”。关于《韵镜》一书的列字，李新魁在《韵镜研究》一文中认为：“《韵镜》二百〇六韵中，共列字三千六百九十五个（据永禄本统计），.”李先生的统计数字与本人的统计数字相差较大，估计李先生没有把多列的同音字，列重了的字或其他情况的字计算在内。孔仲温在《韵镜研究》一书中认为：“，以永禄本言之，其所實之字凡三九三字，………”若加上上面所说的不包括在内的情况（后人注音)，本人的统计数字与孔仲温的统计数字极为接近。  

由于在要比较的韵书中，S2071,P2011，《唐韵》,五代刻本韵书P2014、P2015、P2016、P4747、P5531都是残卷，在比较过程中，对于保存完整的韵，指小韵首字完整，反切至少是切上字完整的韵，我们将这些韵所收之字主要是小韵首字与《韵镜》相对应的韵进行一一对照，既看韵书所收之字主要是小韵首字在《韵镜》中的表现情况，又看《韵镜》所收之字在韵书中的表现情况。而对于那些有残缺的韵，由于我们无法判断《韵镜》里的某个成分在韵书里到底有没有收，因此我们主要将这些残缺的韵里所存小韵一一与《韵镜》相对照。．  

# 第一节 《韵镜》与S2071之比较  

# 2.1.1关于S2071的基本情况  

S2071，就是王国维摹写过的三种切韵残卷的第三种，通常称之为“S2071”,周祖谟在《唐五代韵书集存》中认为其为笺注本切韵。据周先生的书，S2071原卷藏於伦敦博物院，共存三十四页。S2071平声分为平声上与平声下两卷，合上声，去声，入声三卷，共分为五卷。S2071现仅存平声两卷和上声及入声各一卷,而去声一卷全失。  

S2071平声上共有二十六韵，现存三钟至二十六山共二十四个韵。三钟的绝大部分及前面部分残缺，四江至二十六山间有部分残缺。据本人初步统计，S2071平声上小韵完整的（包括韵目及其反切）共有十四个韵，分别为：七之，八微，九鱼，十虞，十一模，十五灰，十六始，十九文，二十殷，二十一元，二十二魂，二十三痕，二十五删，二十六山。S2071平声下共有二十八韵，从一先至二十八凡基本完整，间稍有残缺。前有“切韵卷第二平声下二十八韵”和平声下完整的韵目和反切。据本人统计，S2071平声下小韵完整的（包括韵目及其反切）共有十九个韵，分别为：一先，二仙，三萧，四宵，五肴，六豪，七歌，八麻，九覃，十一阳，十四耕，十八侯，十九幽，二十三蒸，二十四登，二十五咸，二十六衔，二十七严，二十八凡。其中又有十谈缺一切下字，若加上此韵，小韵完整的则共有二十个韵。  

S2071上声共有五十一韵，从一董至五十一范基本完整，间稍有残缺。据本人初步统计，S2071上声小韵完整的（包括韵目及其反切）共有四十四个韵，分别为：一董，二，三讲，四纸，五旨，六止，八语，九麋，十姥，十一，十二蟹，十三骇，十四贿，十五海，十六瑁，十七吻，十八隐，十九阮，二十混，二十一很，二十二旱，二十三潜，二十四产，二十五铣，二十六称，二十七，二十九巧，三十皓，三十一，三十二马，三十三感，三十四敢，三十五养，三十六荡，三十七梗，三十八耿，三十九静，四十三，四十六乔，四十七拯，四十八等，四十九，五十槛，五十一范。上声前也有“切韵卷第三上声五十一韵”字样和上声韵目及反切。  

S2071入声共有三十二韵，从一屋（一屋有残缺）至二十七药（二十七药前半部分有残缺）基本完整，间稍有残缺。入声前有“切韵卷五入声三十二韵”和入声除二十五帖以处的所有韵目及其反切。据本人初步统计，S2071入声小韵完整的（包括韵目及其反切）共有二十个韵，分别为：二沃，三烛，四觉，五质，六物，七，八迄，九月，十没，十一末，十二點，十三，十四屑，十五薛，十六锡，十八麦，十九陌，二十合，二十三，二十四葉。  

S2071共存一百三十个韵（130），二千一百八十二个小韵（2182）。这里的小韵数指保存完整的，即韵目完整，同时反切至少是切上字完整。因此实际所存的小韵数比此数目要多。在所存的这些韵及小韵中，S2071保存完整的，指小韵完整，至少是小韵首字及其反切完整，少数反切仅缺切下字也作完整处理，有九十八个韵，一千六百个小韵。  

据周祖谟的观点，“此书平声上为二十六韵，平声下为二十八韵，上声为五十一韵，入声为三十二韵，韵目的名称和次第等等都同王仁《刊谬初缺切韵》所记陆法言书的面目相同。书中每纽第一字的注文是先出训释，后注反切，反切之后记明一纽的字数，这与前一类的写本在体例上也是一样的。不过同前一类的写本比较，还有不少差别。”。因此周先生认为，“根据以上的比较，可知本书已经不是陆法言的原书。”“……足证本书就是根据陆法言的书而增修的一种《切韵》。”  

另外，根据本人的初步统计，S2071残卷中共有十三处出现了“新加”字样。分别为：二仙，六豪，十六青，二十一盐，三十九静；九月，十没，十五薛（两现），十六锡，十七昔，二十合，二十一韵未二十二洽韵上方。具体出现位置见本人统计资料。  

# 2.1.2《韵镜》与S2071全部韵及其小韵之比较  

经过比较、分析和统计，《韵镜》所列之字与 S2071所收小韵首字之关系主要有以下几种情况：  

# （一）完全相合的情况·  

根据S2071的反切，《韵镜》所列之字与之完全相合的情况共有一千八百一十三例（1813），占S2071所收小韵首字字数的百分之八十三点零九（ $.83.09\%)$ （完全相合指合乎S2071之反切，同时S2071中的字形与《韵镜》中的字形也相同。日本人常将“木”写成“扌”，将“目”写成“耳”等，这里视同完全相合。以下同此。）如：四江韵之小韵，吕江反，依反切为来母江韵。《韵镜》正列於外转第三开（原作外转第三开合，今依杨军《韵镜校笺》一著校正）平声江韵二等来母位置。五支韵之支小韵，章移反，依反切为章母支韵。《韵镜》正列於内转第四开（原作内转第四开合，今依杨军《韵镜校笺》一著校正）平声支韵三等章母位置。  

# （二）《韵镜》所列之字为S2071之非小韵首字  

这种情况是指，《韵镜》所列之字不是S2071中的小韵首字，而是S2071中小韵首字下所收之同音字。这种情况共有一百零二例（102)，占 S2071所收字数的百分之四点六七（ $.4.67\%$ 。如：八微韵之霏小韵，芳非反，依反切为滂（敷)母微韵。《韵镜》於内转第十合平声微韵三等滂（敷）母位置列菲，而菲字S2071在霏小韵下，则此处所列之菲为 S2071之非小韵首字。十虞韵之扶小韵，附夫反，依反切为亚（奉）母虞韵。《韵镜》於内转第十二合（原作内转第十二开合，今依杨军《韵镜校笺》一著校正）平声虞韵三等业（奉）母位置列符，而符字S2071在扶小韵下，则此处所列之符为S2071之非小韵首字。  

# （三〉《韵镜》所列之字为当删之同音字  

这种情况是指，《韵镜》中已列了 S2071中的一个小韵首字，同时又列了该小韵首字下面的一个同音字，此同音字从音韵地位上来说当删。这种情况共有三十九例（39)，占 S2071所收小韵首字字数的百分之一点七九 $\left(1.79\%\right)$ )。 如: 八微韵之祈小韵，渠希反，依反切为群母微韵，下收有同音字顾。《韵镜》已於内转第九开平声微韵三等群母位置列祈，又於内转第十合平声微韵三等群母位置列其同音字顾，则此处所列之顾为当删之同音字。十一阳韵之阳小韵，与章反，依反切为喻母四等阳韵，下收有同音字羊。《韵镜》已於内转第二十一开平声三等阳韵喻母四等位置列阳，又於喻母三等位置列其同音字羊，则此处所列之羊为当删之同音字。  

# 《四）《韵镜》（或S2071）所列为形误字  

这种情况是指《韵镜》所列的字形有误，有很大一部分是S2071所收字形有误，或还有可能是写法问题。这类情况共有四十四例（44），占S2071所收小韵首字字数的百分之二点零二（ $.2.02\%$ 。如：十阳韵之商小韵，书羊反，依反切为书母阳韵。《韵镜》於内转第三十一开平声阳韵三等书母位置列商，商乃商之形误。十四耕韵之小韵，女耕反，依反切为泥（娘）母耕韵。《韵镜》於处转第三十五开平声耕韵二等泥（娘）位置列，乃之形误。  

# 《五）《韵镜》所列之字与S2071的反切不合  

这种情况是指，《韵镜》中收有S2071中的小韵首字，但《韵镜》中所列的位置与S2071反切所反映的位置不合。这类情况共有三十三例（33），占S2071所收小韵首字字数的百分之一点五一： $(1.51\%)$ )。如：七之韵之诗小韵，所之反，依反切为生母之韵。《韵镜》於内转第八开平声之韵三等书母位置列诗，依S2071则位置不合。六止韵之以小韵，羊止反，依反切为喻母四等止韵。《韵镜》於内转第八开上声止韵喻母三等位置列以，依S2071则位置不合。  

《六》《韵镜》所列之字为当删之重字  

这种情况是指《韵镜》中已列了一个与S2071的小韵首字对等的字，但又在某处列了一个与之完全相同的一个字。这样的字从音韵地位上来说当为当删的重字。这类情况共有二十五例（25），占 S2071所收小韵首字字数的百分之一点一五 $(1.15\%)$ ）。如：二十三潜韵之板小韵，布绾反，下收同音字版，依反切为帮母潜韵。《韵镜》已於外转第二十四合上声潜韵二等帮母位置列版，又於外转第二十一开上声产韵二等帮母位置列版，则外转第二十一开所列之版为当删之重字。一先韵之小韵，布玄反，依反切为帮母先韵字。《韵镜》已於外转第二十三开平声先韵四等帮母位置列，又於外转第二十四合平声先韵四等帮母位置列，则外转第二十四合所列之为当删之重字。  

<七〉《韵镜》所列之字字形有误且为当删之同音字

# 2.2.1关于P2011的基本情况  

此书出自敦煌，因此又称为敦煌本王韵。后为伯希和劫去，编号为P2011,现藏於巴黎国家图书馆。此书刘复曾手抄校刊，收入敦煌琐。学术界一般简称之为P2011。据周祖谟在《唐五代韵书集存》一书中的叙述，此书“原物共二十二纸，作两面书，每面多的有三十五六行，每韵另起，韵首一字上有朱书韵次数目，一纽字数也以朱笔书写。”P2011存有平上去入五卷。平声作两卷来处理,但平声的韵目数次不分上下，从残存的五支一直到五十四凡。平声上共二十六韵，从残存的五支到二十五删。韵首及一东、二冬、三钟、四江全残，十四佳、十七真、十八臻、十九文、二十殷、二十一元、二十二魂、二十三痕、二十六山也全残。全部残缺的共有十三个韵。现留存的有十三个韵，留存的十三个韵中没有小韵保存完整的韵。平声下有二十八韵，从二十七先到五十四凡。四十耕、四十二清二韵全残。现留存的有二十六个韵，其中小韵保存完整的有七个韵，分别为：三十二豪，四十三尤，五十登，五十一咸，五十二衔，五十三严，五十四凡。上声共有五十二韵，卷首书有“刊谬初缺切韵卷第三上声五十二韵”，从一董到五十二范。十二蟹，十三骇，三十七梗，三十八耿四韵全残。现留存的有四十八个韵，其中小韵保存完整的有十九个韵，分别为：七尾，八语，十七吻，十八隐，十九阮，二十混，二十一很，三十皓，三十三感，四十二厚，四十三，四十五琰，四十六乔，四十七拯，四十八等，四十九，五十槛，五十一广，五十二范。去声共有五十七韵，卷首书有“刊谬补缺切韵卷第四去声五十七韵”，从一送到五十七梵。二宋、三用、四绛、二十废四韵全部残缺。现留存的有五十三个韵，其中小韵保存完整的有二十一个韵，分别为：七志、八未，十四祭，二十三，二十四，二十五恩，二十六恨，三十四效，三十五号，三十六笛，三十七冯，三十九阙，四十五径，四十八幼，四十九沁，五十艳，五十一榛，五十二證，五十四陷，五十五，五十六严。入声共三十二韵，卷首书有“（刊谬）补缺（切韵卷第五入声三十）二韵”，从一屋到二十四叶。七、二十合，二十五占、二十六缉，二十七药、二十八铎、二十九职，三十德，三十一业，三十二乏十韵全部残缺。现存有二十二个韵，其中小韵保存完整的有三个韵，分别为：四觉，十一末，二十三。由此可见，P2011平声有五十四韵，上声有五十二韵，去声有五十七韵，入声有三十二韵，全书共分为一百九十五韵，比陆法言切韵增加严韵的上声广韵和去声严韵两韵。  

P2011共存有一百六十二（162）个韵，二千零七十三（2073）个小韵。这里的小韵数指保存完整的，即韵目及其反切，至少是切上字完整。因此实际小韵数比此数目要多。在所存的这些韵及小韵中，P2011保存完整的，指小韵完整，至少是小韵首字及其反切完整的情况共有五十（50）个韵，六百六十七（667)个小韵。  

周祖谟在《唐五代韵书集存》一书中认为：“本书平声卷首残阙，上去入三声卷首都题有‘朝议郎行衢州信安县尉王仁字德温新撰定’，则作者为王仁无疑。”同时周先生认为，“王书就陆法言《切韵》刊正伪谬，增字加训，在唐代一定比较流行，所以传本不只一种。故宫博物院所藏宋濂跋本‘刊谬补缺切韵’也是王仁书，但抄写年代比本书晚，反切和注释与本书也不尽相同，文字上的脱误也比较多。”  

周先生在《唐五代韵书集存》一书中又说：“至於反切则改变不多，与第一类《切韵》残本几乎完全相合，而笺注本一则不如此。”因此，周先生认为：“由此可见王韵的反切除去避讳改字的例子以外与第一类《切韵》残本的反切大都相合。那么，他所根据的《切韵》与陆法言书一定非常接近的。”  

# 2.2.2《韵镜》与P2011全部韵及其小韵之比较  

经过比较、分析和统计，《韵镜》所列之字与P2011 所收小韵首字之关系主要有以下几种情况：  

# 〈一）完全相合的情况  

根据P2011的反切，《韵镜》所列之字与之完全相合的情况共有一千七百一十例（1710)，占P2011所收小韵首字字数的百分之八十二点四九 $(82.49\%)$ 。如：五支韵之斯小韵，息移反，依反切为心母支韵。《韵镜》正於内转第四开（原作内转第四开合，今依杨军《韵镜校笺》一著校正）平声支韵四等心母位置列斯。七之韵之思小韵，息兹反，依反切为心母之韵。《韵镜》正於内转第八开平声之韵四等心母位置列思。  

# 《二）《韵镜》所列之字为P2011之非小韵首字  

这种情况是指，《韵镜》所列之字不是P2011中的小韵首字，而是P2011中小韵首字下所收之同音字。这种情况共有一百例（100)，占P2011所列小韵首字字数的百分之四点八二 $(4.82\%)$ 。如：八微韵之霏小韵，芳非反，依反切为滂（敷)母微韵。《韵镜》於内转第十合平声微韵三等滂（敷）母位置列菲。王一菲在霏小韵下。依P2011 则此处所列之菲为非小韵首字。九鱼韵之锄小韵，助鱼反，依反切为崇母鱼韵。《韵镜》於内转第十一开平声鱼韵二等崇母位置列。王一在锄小韵下。依P2011则此处所列之为非小韵首字。  

# 《三〉《韵镜》所列之字为当删之同音字  

这种情况是指，《韵镜》中已列了P2011中的一个小韵首字，同时又列了该小韵首字下面的一个同音字，此同音字从音韵地位上来说当删。这种情况共有十例（10)，占P2011所收小韵首字字数的百分之零点四八（ $.0.48\%$ 。如：十七吻韵之小韵，鱼吻反，下收有同音字，依反切为疑母吻韵。《韵镜》已於外转第二十合上声吻韵三等疑母位置列，又於三等溪母位置列其同音字，则此处所列之依 P2011 为当删之同音字。四十五琰韵之琰小韵，以冉反，下收同音字，依反切为喻母四等琰韵。《韵镜》已於外转第四十开（原作内转第四十合，今依杨军《韵镜校笺》一著校正）上声琰韵四等喻母位置列琰，又於外转第三十九开上声琰韵三等禅母位置列其同音字，则此处所列之類依 P2011为当删之同音字。  

# 《四）《韵镜》所列为形误字  

这种情况是指《韵镜》所列的字形有误，有很大一部分是P2011所收字形有误，或还有可能是写法问题。这类情况共有四十二例（42)，占 P2011 所收小韵首字字数的百分之二点零三！ $\left(2.03\%\right)$ 。如：十三霁韵之契小韵，苦计反，依反切为溪母霁韵。《韵镜》於外转第十三开去声雾韵四等溪母位置列契。契当为契之形误。十七央韵之堇小韵，丑芥反，依反切为彻母央韵。《韵镜》於外转第十三开入声决韵（去声寄此）二等微母位置列蕙。蕙当为之形误。  

# 〈五）《韵镜》所列之字与P2011的反切不合  

这种情况是指，《韵镜》中收有P2011中的小韵首字，但《韵镜》中所列的位置与 P2011反切所反映的位置不合。这类情况共有二十三例（23），占 P2011所收小韵首字字数的百分之一点一一（ $.1.11\%$ ）。如：五十二证韵之孕小韵，以证反，依反切为喻母四等证韵。《韵镜》於内转第四十二开去声证韵喻母三等位置列孕，则与王一的反切不合。三烛韵之欲小韵，余蜀反，依反切为喻母四等烛韵。《韵镜》於内转第二合（原作内转第二开合，今依杨军《韵镜校笺》一著校正）入声烛韵喻母三等位置列欲，则与王一反切不合。  

# 《六）《韵镜》所列之字为当删之重字  

这种情况是指《韵镜》中已列了一个与P2011的小韵首字对等的字，但又在某处列了一个与之完全相同的一个字。这样的字从音韵地位上来说当为当删的重字。这类情况共有六例（6)，占P2011所收小韵首字字数的百分之零点二九$(0.29\%)$ 。如：六至韵之恣小韵，资四反，依反切为精母至韵。《韵镜》已於内转第六开至韵四等精母位置列恣，又於内转第八开去声志韵四等精母位置列恣，则内转第八开所列之恣为当删之重字。五十艳韵所列之染小韵，而赡反，依反切为日母艳韵。《韵镜》已於外转第三十九开去声艳韵三等日母位置列染，又於外转第四十开（原作外转第四十合，今依杨军《韵镜校笺》一著校正）去声艳韵四等日母位置列染，则外转第四十开（原作外转第四十合，今依杨军《韵镜校笺》一著校正）所列之染为当删之重字。  

<七>《韵镜》所列之字为P2011非小韵首字且位置与P2011的反切不合  

这种情况是上述第二种情况和第五种情况的综合。此类情况共有一例（1），占P2011所收小韵首字字数的百分之零点零五 $(0.05\%)$ 。如：二十七翰韵之谐小韵，朝干反，下收同音字赞，依反切为知母翰韵。《韵镜》於外转第二十三开去声翰韵一等精母位置列赞。若依王一，则此位所列之赞为王一非小韵首字且位置与王一的反切不合。  

（八）《韵镜》所列之字与P2011所收之小韵首字为同一字的不同形体  

这类情况共有五十七例（57)，占P2011所收小韵首字字数的百分之二点七五 $(2.75\%)$ 。如：二十四寒韵之小韵，仓干反，依反切为清母寒韵。《韵镜》於外转第二十三开平声寒韵一等清母位置列餐。“”与“餐”当为同一字之不同形体。三十四麻韵之小韵，食遮反，依反切为船母麻韵。《韵镜》於外转第二十九开（原作内转第二十九开，今依杨军《韵镜校笺》一著校正）平声麻韵三等船母位置列蛇，“”与“蛇”当为同一字之不同形体。  

# （九）同一音韵地位的字  

这种情况是指，在《广韵》或其他韵书中 A小韵首字下收有B 同音字，而P2011用B作小韵首字，而B下不见A，《韵镜》所列的也是A。这种情况共有三十八（38）例，占P2011所收小韵首字字数的百分之一点八三 $\cdot(1.83\%)$ 。如：十九阮韵之偃小韵，於反，依反切为影母阮韵。《韵镜》於外转第二十一开上声阮韵三等影母位置列堰。《广韵》堰在阮韵偃小韵下，P2011阮韵偃小韵下未收堰。“偃”与“堰”当为同一音韵地位的字。四十二厚韵之取小韵，仓垢反，依反切为清母厚韵。《韵镜》於内转第三十七开厚韵一等清母位置列趣。《广韵》取在趣小韵下，P2011取小韵下未收趣。“取”与“趣”当为同音韵地位的字。  

《十）《韵镜》所列之字字形有误且为非小韵首字  

这种情况是第二种情况和第四种情况的综合。这类情况共有四例（4)，占P2011所收小韵首字字数的百分之零点一九 $(0.19\%)$ 。如：五十登韵之柩小韵，古恒反，依反切为见母登韵。《韵镜》於内转第四十二开平声登韵一等见母位置列，此处所列之当为字之形误，且字 S2071 在椭小韵下。依 P2011 则此处所列之是字形有误且为非小韵首字。三十六个韵（《韵镜》个韵和过韵）之嬴小韵，郎过反，下收有同音字，依反切为来母个韵（《韵镜》个韵和过韵)。《韵镜》於内转第二十八合去声过韵一等来母位置列。依P2011则此位所列之当为之形误且为非小韵首字。  

〈十一〉《韵镜》所列之字与P2011所收之小韵首字为同一音韵地位的字，且与P2011的反切位置不合  

这种情况是上述第五种情况和第九种情况的综合。此类情况共有一例(1),占P2011所收小韵首字字数的百分之零点零五 $(0.05\%)$ 。如：十四祭韵之小韵，楚崴反，依反切为初母祭韵。《韵镜》於外转第十四合去声祭韵三等昌母位置列鑫。《广韵》囊在桑小韵下，P2011桑小韵未收。则“囊”与“桑”当为同一音韵地位的字，且与P2011的反切位置不合。  

《十二）《韵镜》所列字形有误且为同一音韵地位的字  

这种情况是上述第四种情况和第九种情况的综合。此类情况共有一例(1),占P2011所收小韵首字字数的百分之零点零五 $(0.05\%)$ 。如：五十艳韵之小韵，充艳反，依反切为昌母艳韵。《韵镜》於外转第三十九开去声艳韵三等昌母位置列。《广韵》在小韵下，P2011小韵下未列。“”与“”当为同一音韵地位的字，且此处所列之“”当为之形误。  

# 〈十三）《韵镜》列有，而P2011未收  

这种情况是指《韵镜》列有该字，但P2011没有收，或P2011该韵没有收（该字可能在其他韵里出现，由于初步统计，不排除这种情况存在)。这类情况共有五十四例（54)，占P2011所收小韵首字字数的百分之二点六零（ $:2.60\%$ )。如：《韵镜》於外转第三十九开去声陷韵三等疑母位置列，P2011.五十四陷韵未收。《韵镜》於外转第十八合去声恩韵一等精母位置列，P2011二十五恩韵未收。  

〈十四）P2011收有，而《韵镜》未列  

这种情况是指P2011收有该小韵首字，但《韵镜》里相应位置没有列出，在其他地方也没有出现。这类情况共有九十六例（96)，占 P2011所收小韵首字字数的百分之四点六三 $(4.63\%)$ 。如：四十三尤韵之惆小韵，去愁反，依反切为溪母尤韵。《韵镜》於内转第三十七开平声尤韵三等溪母位置列丘（原缺末一笔)。丘字P2011 尤韵去求反，而未列惆，其他位置亦未列。四十三韵之小韵，慈纠反，依反切为欲母韵。《韵镜》於内转第三十七开上声韵四等从母位置列有韵之湫小韵，而未列默韵之小韵。

# 2.2.3《韵镜》与P2011保存完整的韵及其小韵之比较  

如前统计，P2011 保存完整的有五十个韵，六百六十七个小韵。现将P2011  

保存完整的韵及其小韵与《韵镜》相比较，情况如下：  

# 〈一）完全相合的情况  

这种情况共有五百五十七例，占P2011保存完整的韵所收小韵首字字数的百分之八十三点五一（ $:83.51\%)$ 。如：三十二豪韵之高小韵，古劳反，依反切为见母豪韵。《韵镜》正於外转第二十五开平声豪韵一等见母位置列高。四十二尤韵之柔小韵，耳由反，依反切为日母尤韵。《韵镜》正於内转第三十七平声尤韵三等日母位置列柔。  

# 〈二〉《韵镜》所列之字为P2011之非小韵首字

# 第三节 《韵镜》与“王三”之比较  

# 2.3.1关于“王三”的基本情况  

“王三”一书共有一百九十五韵。其中平声五十四韵，（平声不分上下，若按上下来分，上平声二十六韵，下平声二十八韵）上声五十二韵，去声五十七韵，入声三十二韵。  

据本人统计，“王三”一书共收有三千六百三十一（3631）个小韵首字。其中平声一千零九十二（1092）个[不分上下，若按上下来分，上平声五百三十四（534）个，下平声五百五十八个（558）】，上声八百五十三（853）个，去声一千零四十六（1046）个，入声六百四十（640）个。（其中去声四十九沁韵有两个小韵缺反语；去声五十六韵的韵目及其反语缺；入声六物有一小韵缺反语；入声十三有一小韵注释在反切前，当为后人所增；入声十八麦韵有两小韵漫难辨。这些残缺或难辨处据龙宇纯《唐写本王仁刊谬补缺切韵校笺》补，另有少数地方据此更正。具体情况参看本人的统计资料。)  

“王三”一书前有“王仁‘刊谬补缺切韵序’”；“陆词字法言撰切韵序";“王三”正文分为五卷（平声一卷，又为两卷)，每一卷前面指出该卷所包含的韵数，所收的字数，然后列出所包含的各韵韵目，并注出反切及各家韵书的异同，反切用“xx反”，最后按韵次排列各韵的情况。每一韵的上方用数字“一、二、·…"注出其韵次，每一韵之间留有空白。每一个小韵於第一个字下先注出其反切，然后是注释，最后是标出各个小韵所收同音字的字数。每一个小韵依次相连，没有间隔。有少数小韵的注释在反切之前，当为后人所增。全书共收一百九十五韵，其中去声严韵韵前列有韵目，“严鱼淹反陆无此韵目失”，后面正文缺。“王三”一书基本完整，有少数地方缺失或漫不清。“王三”卷未有宋濂记。  

# 2.3.2《韵镜》与“王三”之比较  

经过比较、分析和统计，《韵镜》所列之字与“王三”所收小韵首字之关系主要有以下几种情况：  

# 《一）完全相合的情况  

根据“王三”的反切，《韵镜》所列之字与之完全相合的情况共有二千九百五十五例（2955)，占《韵镜》所收字数的百分之七十五点九零 $(75.90\%)$ ，占“王三”所收小韵首字字数的百分之八十一点三八 $(81.38\%)$ 。（完全相合指合乎“王三”之反切，同时“王三”中的字形与《韵镜》中的字形也相同。日本人常将“木”写成“扌”，将“目”写成“且”等，这里视同完全相合。）如：一东韵之同小韵，徒红反，依反切为定母东韵。《韵镜》正於内转第一开平声东韵一等定母位置列同。二冬韵之攻小韵，古冬反，依反切为见母冬韵。《韵镜》正於内转第二合（原作内转第二开合，今依杨军《韵镜校笺》一著校正）平声冬韵一等见母位置列攻。  

《二〉《韵镜》所列之字为“王三”之非小韵首字  

这种情况是指，《韵镜》所列之字不是“王三”中的小韵首字，而是“王三”中小韵首字下所收之同音字。这种情况共有一百九十二例（192)，占《韵镜》所列字数的百分之四点九三 $(4.93\%)$ ，占“王三”所列小韵首字字数的百分之五点二九 $(5.29\%)$ 。如：八微韵之斐小韵，匪肥反，下收同音字非，依反切为帮（非)母微韵。《韵镜》於内转第十合平声微韵三等帮（非）母位置列非。依“王三”则此处所列之非为非小韵首字。九鱼韵之锄小韵，助鱼反，下收同音字，依反切为崇母鱼韵。《韵镜》於内转第十一开平声鱼韵二等崇母位置列。依“王三”则此处所列之为非小韵首字。  

# 《三〉《韵镜》所列之字为当删之同音字  

这种情况是指，《韵镜》中已列了“王三”中的一个小韵首字，同时又列了该小韵首字下面的一个同音字，此同音字从音韵地位上来说当删。这种情况共有八十六例（86)，占《韵镜》所列字数的百分之二点二一（ $.2.21\%)$ ，占“王三”所收小韵首字字数的百分之二点三七（ $.2.37\%$ 。如：三十七阳韵之阳小韵，与章反，下收同音字羊，依反切为喻母四等阳韵。《韵镜》已於外转第三十一开平声阳韵四等喻母位置列阳，又於喻母三等位置列羊，则此处所列之羊为当删之同音字。四十清韵之精小韵，子清反，下收同音字菁，依反切为精母清韵。《韵镜》已於外转第三十三开平声清韵一等精母位置列精，又於外转第三十五开平声青韵四等精母位置列菁。则此处所列之菁为当删之同音字。  

# 《四〉《韵镜》所列为形误字  

这种情况是指《韵镜》所列的字形有误，有很大一部分是“王三”所收字形有误，或还有可能是写法问题。这类情况共有一百一十六例（116)，占《韵镜》所列字数的百分之二点九八（ $.2.98\%)$ ，占“王三”所收小韵首字字数的百分之三点一九 $(3.19\%)$ 。如：四十二青韵之宁小韵，奴丁反，依反切为泥母青韵。《韵镜》於外转第三十五开平声青韵四等泥母位置列，当为宁字之形误。一董韵之小韵，力董反，依反切为来母董韵。《韵镜》於内转第一开上声董韵一等来母位置列，当为曬之形误。  

# 〈五〉《韵镜》所列之字与“王三”的反切不合  

这种情况是指，《韵镜》中收有“王三”中的小韵首字，但《韵镜》中所列的位置与“王三”反切所反映的位置不合。这类情况共有四十三例（43），占《韵镜》所列字数的百分之一点一零（ $.1.10\%)$ ，占“王三”所收小韵首字字数的百分之一点一八 $\left(1.18\%\right)$ 。如：六止韵之以小韵，羊止反，依反切为喻母四等止韵。《韵镜》於内转第八开上声止韵三等喻母位置列以，与“王三”的反切不合。十三霁韵之丽小韵，鱼帝反，依反切为疑母雾韵。《韵镜》於外转第十三开去声雾韵四等来母位置列丽，与“王三”的反切不合。  

《六〉《韵镜》所列之字为当删之重字  

这种情况是指《韵镜》中已列了一个与“王三”的小韵首字对等的字，但又在某处列了一个与之完全相同的一个字。这样的字从音韵地位上来说当为当删的重字。这类情况共有三十例(30)，占《韵镜》所列字数的百分之零点七七 $(0.77\%)$ ?占“王三”所收小韵首字字数的百分之零点八三 $(0.83\%)$ ）。如：四十一清韵之轻小韵，去盈反。依反切为溪母清韵。《韵镜》已於外转第三十三开平声清韵四等溪母位置列轻，又於外转第三十五开平声青韵四等溪母位置列轻，则外转第三十五开所列之轻为当删之重字。二十八仙韵之甄小韵，居延反，依反切为见母仙韵。《韵镜》已於外转第二十一开平声仙韵四等见母位置列甄，又於外转第二十三开平声仙韵三等见母位置列甄，则外转第二十三开所列之甄为当删之重字。  

〈七〉《韵镜》所列之字为“王三”非小韵首字且位置与“王三”的反切不合  

这种情况是上述第二种情况和第五种情况的综合。此类情况共有两例(2),占《韵镜》所列字数的百分之零点零五 $(0.05\%)$ ，占“王三”所收小韵首字字数的百分之零点零六 $(0.06\%)$ 。如：二冬韵之小韵，先恭反，下收同音字淞，依反切为心母冬韵。《韵镜》於内转第二合（原作内转第二开合，今依杨军《韵镜校笺》一著校正）平声钟韵四等心母位置列淞。若依“王三”，些外所列之淞为“王三”非小韵首字且位置与“王三”的反切不合。十四祭韵之祭小韵，子例反，下收同音字德，依反切为精母祭韵。《韵镜》於外转第十六合去声泰韵一等喻母位置列德，若依“王三”，此处所列之德为“王三”非小韵首字且位置与“王三”反切不合。  

〈八）《韵镜》所列之字字形有误且为当删之同音字  

这种情况是第三种情况和第四种情况的综合。这类情况共有二例(2)，占《韵镜》所列字数的百分之零点零五（ $:0.05\%.$ ），占“王三”所收小韵首字字数的百分之零点零六（ $.0.06\%$ 。如：一东韵之融小韵，余隆反，下收同音字彤，依反切为喻母四等东韵。《韵镜》已於内转第一开平声东韵四等喻母位置融，又於喻母三等位置列彤。“王三”彤在融小韵下，注释为“祭名，又敕林反”。《广韵》彤在融小韵下，注释为“祭名，又敕林反”。按注释为祭名，则字形当从月作彤。若依“王三”，则此处所列之彤为字形有误且为当删之同音字。如：二沃韵之瑁小韵，莫沃反，下收同音字媚，依反切为明母沃韵。《韵镜》已於内转第二合（原作开合，今依杨军《韵镜校笺》一著校正）入声沃韵一等明母位置列瑁，又於烛韵三等明（微）母位置列婿。颇疑此处所列之婿即为媚之形误，若此则此处所列之为字形有误且为当删之同音字。  

这类情况共有六十七例(67)，占《韵镜》所列字数的百分之一点七二 $(1.72\%)$ 占“王三”所收小韵首字字数的百分之一点八五（ $.1.85\%$ 。如：五支韵之弥小韵，武移反，依反切为明（微）母支韵。《韵镜》於内转第四开（原作开合，今依杨军《韵镜校笺》一著校正）平声支韵四等明母位置列彌，“弥”、“彌”当为同一字的不同形体。《广韵》即作彌，下收弥，注云“上同”。十七真韵之小韵，陟粼反，依反切为知母真韵。《韵镜》於外转第十七开平声真韵三等知母位置列珍。“”、“珍”当为同一字的不同形体。《广韵》即作珍，注云俗作。  

# （十）同一音韵地位的字  

这种情况是指，在《广韵》中A小韵首字下收有B同音字，而“王三”用B作小韵首字，而B下不见A，《韵镜》所列的也是A。这种情况共有六十六（66)例，占《韵镜》所列字数的百分之一点七零 $(1.70\%)$ ，占“王三”所收小韵首字字数的百分之一点八二（ $.1.82\%$ 。如：二十八仙韵之便小韵，房连反，依反切为亚（奉）母仙韵。《韵镜》於外转第二十一开平声仙韵四等母位置列。《广韵》在便小韵下，“王三”便小韵下未收。“便”、“梗”当为同一音韵地位的字。三十一肴韵（《韵镜》作交）之嘲小韵，张交反，依反切为知母肴韵。《韵镜》於外转第二十五开平声交韵二等知母位置列。《广韵》在嘲小韵下，“王三”嘲小韵下未收。“嘲”、“”当为同一音韵地位的字。  

# 〈十一〉《韵镜》所列之字字形有误且为非小韵首字  

这种情况是第二种情况和第四种情况的综合。这类情况共有七例（7)，占《韵镜》所列字数的百分之零点一八 $(0.18\%)$ ），占“王三”所收小韵首字字数的百分之零点一九 $(0.19\%)$ 。如：五支韵之小韵，式支反，下收同音字施，依反切为书母支韵。《韵镜》於内转第四开（原作内转第四开合，今依杨军《韵镜校笺》一著校正）平声支韵三等书母位置列。此处所列之囊当为施之形误，且施字S2071在小韵下，则此处所列之囊是字形有误且为非小韵首字。六脂韵之馨小韵，渠脂反，下收同音字耆，依反切为群母脂韵。《韵镜》於内转第六开平声脂韵三等群母位置列耆。此处所列之耆为耆之形误且为非小韵首字。  

《十二）《韵镜》所列之字字形有误且与“王三”的反切不合  

这种情况是第四种情况和第五种情况的综合。这种情况共有一例(1)，占《韵镜》所列字数的百分之零点零三（ $.0.03\%)$ ，占“王三”所收小韵首字字数的百分之零点零三 $(0.03\%)$ 。如：二十八仙韵之小韵，於权反。当为之形误。《韵镜》於外转第二十四合平声仙韵三等晓母位置列，此处所列之当为之形误且与“王三”的反切不合。  

<十三〉《韵镜》所列之字字形有误且为当删之重字  

这种情况是第四种情况和第六种情况的综合。这类情况共有一例(1)，占《韵镜》所收字数的百分之零点零三 $(0.03\%)$ ，占“王三”所收小韵首字字数的百分之零点零三 $\langle0.03\%\rangle$ 。如：四十耕韵之小韵，女耕反，依反切为泥（娘）母耕韵。《韵镜》已於外转第三十五开平声耕韵二等泥（娘）母位置列，偏当为之形误，又於外转第三十三开平声庚韵二等泥（娘）母位置列窟，则外转第三十三开所列之偏为字形有误且为当删之重字。  

# 〈十四）《韵镜》所列当为据“王三”又音所列  

此类情况共有八例（8)，占《韵镜》所列字数的百分之零点二二（ $.0.22\%)$ 占“王三”所收小韵首字字数的百分之零点二一 $(0.21\%)$ 。如：二冬韵之彤小韵，徒冬反，依反切为定母冬韵，下收同音字，且下注云“又他冬切”。《广韵》冬韵即收有小韵，他冬切，依反切为透母冬韵。《韵镜》於内转第二合（原作内转第二开合，今依杨军《韵镜校笺》一著校正）平声冬韵一等定母位置列彤，透母位置列。若依“王三”，此处所列之为当删之同音字或是据又音所列,此处理为据“王三”又音所列。七尾韵之小韵，许伟反，依反切为晓母尾韵，下收同音字，注云又鱼岂反。《广韵》即收有小韵，鱼岂切，依反切为疑母尾韵。《韵镜》於内转第十合上声尾韵三等晓母位置列，於内转第九开上声尾韵三等疑母位置列，若依“王三”，此处所列之为当删之同音字或是据又音所列，此处理为据“王三”又音所列。  

<十五〉“王三”中所列小韵首字为后人所增加的字  

这种情况是指一般位于“王三”某一韵的韵末，且字体或书写格式及体例与其他部分不同，依龙宇纯在《唐写全本王仁刊谬补缺切韵校笺》一著中的意见是后人增加的小韵字。这种情况共有四例（4)，占《韵镜》所列字数的百分之零点一零( $.0.10\%)$ ），占“王三”所列小韵首字字数的百分之零占一一 $(0.11\%)$ 。如：三十九庚韵之小韵，古行反。据龙宇纯《唐写全本王仁刊谬补缺切韵校笺》此当为后人所增。四十三尤韵之衷小韵，蒲沟反。据龙宇纯《唐写全本王仁刊谬补缺切韵校笺》此当为后人所增。  

<十六〉《韵镜》所列之字与“王三”所收之小韵首字为同一字的不同形体，且与“王三”的反切位置不合  

这种情况是上述第五种情况和第九种情况的综合。此类情况共有一例（1)，占《韵镜》所收字数的百分之零点零三 $(0.03\%)$ ），占“王三”所收小韵首字字数的百分之零点零三 $:0.03\%)$ 。如：二十六韵之趁小韵，居展反（按此处之居当为尼之形误，今依原文不作改动)，依反切为见母韵。《韵镜》於外转第二十三开上声韵三等泥（娘）母位置列趁。“趁”、“”当为同一字的不同形体，若依“王三”原文不改则且与“王三”的反切位置不合。  

〈十七〉同一音韵地位的字，且与“王三”的反切位置不合  

这种情况是上述第五种情况和第十种情况的综合。此类情况共有一例（1),占《韵镜》所收字数的百分之零点零三 $(0.03\%)$ ，占“王三”所收小韵首字字数的百分之零点零三（ $.0.03\%$ ）。如：十四祭韵之桑小韵，楚崴反，依反切为初母祭韵。《韵镜》於外转第十四合去声祭韵三等昌母位置列。《广韵》在桑小韵下，“王三”桑小韵下未列。“囊”、“鬓”当为同一音韵地位的字，且与“王三”的反切位置不合。  

〈十八〉《韵镜》列有，而“王三”未收  

这种情况是指《韵镜》列有该字，但“王三”没有收，或“王三”该韵没有收（该字可能在其他韵里出现，由于初步统计，不排除这种情况在存在)。这类情况共有三百-十一例（311)，占《韵镜》所列字数的百分之七点九九 $(7.99\%)$ 占“王三”所收小韵首字字数的百分之八点五七 $(8.57\%)$ 。如：《韵镜》於外转第三开（原作外转第三开合，今依杨军《韵镜校笺》一著校正）平声江韵二等影母位置列映，“王三”江韵未收此小韵。《韵镜》於外转第二十九开（原作内转第二十九开，今依杨军《韵镜校笺》一著校正）平声麻韵四等位置列些，“王三”麻韵未收此小韵。  

# 〈十九）“王三”收有，而《韵镜》未列  

这种情况是指“王三”收有该小韵首字，但《韵镜》里相应位置没有列出，在其他地方也没有出现。这类情况共有一百七十三例（173)，占“王三”所收小韵首字字数的百分之四点七六 $(4.76\%)$ ），占《韵镜》所列字数的百分之四点四四$(4.44\%)$ 》。如：五支韵之小韵，香支反，依反切为晓母支韵。《韵镜》於内转第四开（原作内转第四开合，今依杨军《韵镜校笺》一著校正）平声支韵未列。六脂韵之帷小韵，消悲反，依反切为匣（喻三）母脂韵。《韵镜》於内转第六开平声脂韵未列。

# 第四节 《韵镜》与《唐韵》残卷之比较  

# 2.4.1关于《唐韵》残卷的基本情况  

《唐韵》残卷存去入两卷（第四卷、第五卷)。去声卷共有五十九韵，但多有残缺，从韵首至八未前半部分全部残缺，从十九代后半至二十五愿前半也全部残缺，在所存的韵中又间有残缺。据本人初步统计，去声卷小韵完整的（包括韵目及其反切）共有十四个韵，分别为：十一暮，十五卦，十六怪，二十六恩，二十七恨，三十一，三十四啸，三十八个，三十九过，五十一幼，五十三艳，五十四搽，五十五证，五十八。入声卷共有三十四韵，卷首书有“唐韵卷第五入声三十四韵”，从韵首至韵尾韵目完整，只是中间间有残缺。据本人的初步统计，入声卷小韵完整的（包括韵目及其反切）共有十个韵，分别为：八，九迄，十月，十五，十八锡，二十三，二十四洽，二十五，二十九药，三十四乏。（三烛当全，但有据补部分；二十麦缺一切下字，若加上此二韵则为十二个韵。)  

《唐韵》残卷韵与韵之间留有空隙或分列书写，韵目上不加朱圈。每纽於第一字下先出训释，后出反切，最后标出该纽所收同音字的字数（注意入声乏韵与此体例不同，乏韵於每一纽第一字下先出反切，后出训释，最后标出本纽所收之同音字的字数)。每一纽若有新加字，则注明“x加y”，前者表示包括后者在内的总字数，后者表示新增加的字数，同时在新增加的字后也注明“加”字。所收之字基本上都有训释，也有的常用字只注音切而无训释。在又音方面，本书除用反切表示又音外，还用了大量的直音。这与王韵有别，而与广韵相似。  

《唐韵》共存八十一个韵，一千二百八十一个小韵。这里的小韵数指保存完整的，即韵目及其反切，至少是切上字完整。因此实际小韵数比此数目要多。在所存的这些韵及小韵中，保存完整的，指小韵完整，至少是小韵首字及其反切完整，少数反切仅缺切下字也作完整处理，有二十五个韵，三百一十九个小韵。  

周祖谟在《唐五代韵书集存》里说：“本书分韵与切韵不同的地方在於去声分为五十九韵，比切韵多、换、过三韵；入声分为三十四韵，比切韵多术葛二韵。去声震分为两韵，翰换分为两韵，个过分为两韵，入声质术分为两韵，歇未分为两韵。由此可推，平声真韵后当有淳韵，寒韵后当有桓韵，歌韵后当有戈韵，上声也一定与平去入三声相应。这在切韵王韵之外另成一个系统。”据周先生的意见，“本书名为《唐韵》，但还不能认为就是孙的原作。”周先生认为本书为孙书之后的增修本之一种，同时认为本书去孙原书还不会太远，本书可能是比较接近於孙恤原书的一种增修本。  

# 2.4.2《韵镜》与《唐韵》残卷全部韵及其小韵之比较  

经过比较、分析和统计，《韵镜》所列之字与《唐韵》残卷所收小韵首字之关系主要有以下几种情况：  

# （一）完全相合的情况  

根据《唐韵》的反切，《韵镜》所列之字与之完全相合的情况共有一千零七十六例（1076)，占《唐韵》所收小韵首字的百分之八十四点零零 $(84.00\%)$ 。如：十遇韵之付小韵，方遇反，依反切为帮（非）母遇韵。《韵镜》正於内转第十二合（原作内转第十二开合，今依杨军《韵镜校笺》一著校正）去声遇韵三等帮（非）母位置列付。十一暮韵之路小韵，洛故反，依反切为来母暮韵。《韵镜》正於内转第十二合（原作内转第十二开合，今依杨军《韵镜校笺》一著校正）去声暮韵一等来母位置列路。  

# 《二〉《韵镜》所列之字为《唐韵》之非小韵首字  

这种情况是指，《韵镜》所列之字不是《唐韵》中的小韵首字，而是《唐韵》中小韵首字下所收之同音字。这种情况共有四十四例（44)，占《唐韵》所收小韵首字字数的百分之三点四三 $(3.43\%)$ 。如：二十九换韵之叛小韵，薄半反，依反切为母换韵，下收同音字畔。《韵镜》於外转第二十四合去声换韵一等母位置列畔，则此处所列之畔为《唐韵》之非小韵首字。三十二霰韵之现小韵，胡甸反，下收同音字见，依反切为匣母霰韵。《韵镜》於外转第二十三开去声霰韵四等匣母位置列见，则此处所列之见为《唐韵》之非小韵首字。·  

# 《三）《韵镜》所列之字为当删之同音字  

这种情况是指，《韵镜》中已列了《唐韵》中的一个小韵首字，同时又列了--个该小韵首字下面的一个同音字，此同音字从音韵地位上来说当删。这种情况共有四例（4)，占《唐韵》所收小韵首字字数的百分之零点三一 $(0.31\%)$ 。如：三烛韵之束小韵，书玉反，下收同音字，依反切为书母烛韵。《韵镜》已於内转第二合（原作内转第二开合，今依杨军《韵镜校笺》一著校正）入声烛韵三等书母位置列束，又於二等初母位置列，则此处所列之为当删之同音字。二十九药韵之芍小韵，张略反，下收同音字著，依反切为知母药韵。《韵镜》已於内转第三十一开入声药韵三等知母位置列芍（字形稍有误)，又於澄母位置列著，依《唐韵》则此处所列之著为当删之同音字。（《唐韵》著有又音，又音直略反，  

依又音正可列於此位。)  

# 《四〉《韵镜》所列为形误字  

这种情况是指《韵镜》所列的字形有误，有的也有可能是《唐韵》所收字形有误，或还有可能是写法问题。这类情况共有三十四例（34)，占《唐韵》所收小韵首字字数的百分之二点六五 $(2.65\%)$ 。如：十三霁韵之帮小韵，苦计反。帮当为契之不同形体或不同写法。依反切为溪母霁韵。《韵镜》於外转第十三开去声雾韵四等溪母位置列契，依《唐韵》此处所列之契为形误字。十七央韵之堇小韵，丑介反，依反切为彻母央韵。《韵镜》於外转第十三开入声（去声寄此）夫韵二等彻母位置列蕙，当为之形误。  

# 〈五〉《韵镜》所列之字与《唐韵》的反切不合  

这种情况是指，《韵镜》中收有《唐韵》中的小韵首字，但《韵镜》中所列的位置与《唐韵》反切所反映的位置不合。这类情况共有二十二例（22)，占《唐韵》所收小韵首字字数的百分之一点七二· $(1.72\%)$ 。如：十九代韵之载小韵，才代反，依反切为从母代韵。《韵镜》於外转第十三开去声代韵一等精母位置列载。依《唐韵》则位置不合。四十三漾韵之逛小韵，居况反，依反切为见母漾韵。《韵韵》於内转第三十二合去声漾韵三等群母位置列逛。依《唐韵》则位置不合。  

# 《六）《韵镜》所列之字为当删之重字  

这种情况是指《韵镜》中已列了一个与《唐韵》的小韵首字对等的字，但又在某处列了一个与之完全相同的一个字。这样的字从音韵地位上来说当为当删的重字。这类情况共有六例（6)，占《唐韵》所收小韵首字字数的百分之零点四七$(0.47\%)$ 。如：一屋韵之郁小韵，於六反，依反切为影母屋韵。《韵镜》已於内转第一开入声屋韵影母位置列郁，又於内转第二合（原作内转第二开合，今依杨军《韵镜校笺》一著校正）入声烛韵三等影母位置列郁。则内转第二合（原作内转第二开合，今依杨军《韵镜校笺》一著校正）所列之郁为当删之重字。十四韵之杀小韵，所八反，依反切为生母點韵。《韵镜》已於外转第二十三开入声點韵二等生母位置列杀，又於外转第二十一开入声韵二等生母位置列杀，则外转第二十一开所列之杀为当删之重字。  

<七〉《韵镜》所列之字字为《唐韵》非小韵首字且位置与《唐韵》的反切不合  

这类情况共有一例（1)，占《唐韵》所收小韵首字字数的百分之零点零八$\left(0.08\%\right)$ )。如：四十四宕韵之小韵，亦扩反（亦当为乎之形误，今依原文不作改动)，下收同音字潢，依反切为喻母四等宕韵。《韵镜》於内转第三十二合去声宕韵一等匣母位置列潢。若依《唐韵》则此处所列之潢为《唐韵》非小韵首字且位置与《唐韵》的反切不合。  

（八）《韵镜》所列之字与《唐韵》所收之小韵首字为同一字的不同形体  

这类情况共有三十七例（37)，占《唐韵》所收小韵首字字数的百分之二点八九 $(2.89\%)$ 。如：九御韵之筋小韵，倔反，依反切为澄母御韵。《韵镜》於内转第十一开去声御韵三等澄母位置列箸。“筋”、“箸”当为同一字的不同形体。王三即作筋，注云“或作箸”。《广韵》作箸，下收筋，注云“上同”。十二泰韵之小韵，苦盖反，依反切为溪母泰韵。《韵镜》於外转第十五开去声泰韵一等溪母位置列磕。“磕”“”当为同一字的不同形体。《唐韵》作磕，注云“”。《广韵》作磕，注云“磕”  

# 《九）《韵镜》所列之字字形有误且为非小韵首字  

这种情况是第二种情况和第四种情况的综合。这类情况共有一例（1)，占《唐韵》所收小韵首字字数的百分之零点零八： $(0.08\%)$ 。如：十七薛韵之小韵，芳劣反，依反切为滂（敷）母薛韵，下收同音字鳖。《韵镜》於外转第二十一开入声薛韵四等母位置列警。警当为警之形误且为非小韵首字。  

# <十）同一音韵地位的字  

这种情况是指，在《广韵》或其他韵书中A小韵首字下收有B 同音字，而《唐韵》用B作小韵首字，而B下不见A，《韵镜》所列的也是A。这种情况共有十八（18）例，占《唐韵》所收小韵首字字数的百分之一点四一 $(1.41\%)$ 。如：一屋韵之蓄小韵，许六反，依反切为晓母屋韵。《韵镜》於内转第一开入声屋韵三等晓母位置列畜。“蓄”、“畜”当为同一音韵地位的字。王三、《广韵》皆作蓄，下收有畜。《唐韵》蓄小韵下未收畜。一屋韵之小韵，丑六反，依反切为彻母屋韵。《韵镜》於内转第一开入声屋韵三等彻母位置列蓄。“”、“蓄”当为同一字之不同形体。《广韵》作蓄，丑六反，下收有，注云“上同”。《唐韵》小  

韵下未收蓄。  

《十一）《韵镜》所列当为同一字的不同形体且为非小韵首字  

这种情况是上述第二种情况和第八种情况的综合。此类情况共有一例（1),占《唐韵》所收小韵首字字数的百分之零点零八（ $.0.08\%)$ 。十遇韵之煦小韵，香句反，依反切为晓母遇韵，下收同音字煦。《韵镜》於内转第十二合（原作内转第十二开合，今依杨军《韵镜校笺》一著校正)去声遇韵三等晓母位置列响。“煦”“响”当为同一字之不同形体。《唐韵》煦下注云“吐沫”。《广韵》作，下收煦，注云“上同”，同时收有响，注云“吐沫”。则此处所列之响与煦当为同一字的不同形体且为非小韵首字。  

# 〈十二〉《韵镜》列有，而《唐韵》未收  

这种情况是指《韵镜》列有该字，但《唐韵》没有收，或《唐韵》该韵没有收（该字可能在其他韵里出现，由于初步统计，不排除这种情况在存在)。这类情况共有五十二例(52)，占《唐韵》所收小韵首字字数的百分之四点零六 $\left(4.06\%\right)$ 如：《韵镜》於内转第二十七开（原作内转第二十七开合，今依杨军《韵镜校笺》一著校正）去声个韵一等透母位置列拖，《唐韵》三十八个韵未收拖小韵。《韵镜》於外转第十五开去声卦韵二等帮母位置列薛，《唐韵》十五卦韵未收有小韵。  

〈十三〉《唐韵》收有，而《韵镜》未列  

这种情况是指《唐韵》收有该小韵首字，但《韵镜》里相应位置没有列出，在其他地方也没有出现。这类情况共有四十七例（47)，占《唐韵》所收小韵首字字数的百分之三点六七 $(3.67\%)$ 。如：八未韵收有衣小韵，於既反，依反切为影母未韵。《韵镜》於内转第九开去声未韵三等影母位置未列。十四祭韵收有祭小韵，子例反，依反切为精母祭韵。《韵镜》於外转第十五开去声祭韵四等精母位置未列。。

# 2.4.3《韵镜》与《唐韵》保存完整的韵及其小韵之比较  

如前统计，《唐韵》保存完整的有二十五个韵，三百一十九个小韵。现将《唐韵》保存完整的韵及其小韵与《韵镜》相比较，情况如下：  

根据《唐韵》的反切，《韵镜》所列之字与之完全相合的情况共有二百七十七例（277)，占《唐韵》所收小韵首字的百分之八十六点八三（ $86.83\%$ 。如：十一暮韵之路小韵，洛故反，依反切为来母暮韵。《韵镜》正於内转第十二合（原作内转第十二开合，今依杨军《韵镜校笺》一著校正）去声暮韵一等来母位置列路。十五卦韵之隘小韵，乌懈反，依反切为影母卦韵。《韵镜》正於外转第十五开去声卦韵二等影母位置列隘。  

# 《二）《韵镜》所列之字为《唐韵》之非小韵首字  

这种情况是指，《韵镜》所列之字不是《唐韵》中的小韵首字，而是《唐韵》中小韵首字下所收之同音字。这种情况共有九例（9)，占《唐韵》所收小韵首字字数的百分之二点八二（ $\left(2.82\%\right)$ 。如：十一暮韵之小韵，汤故反，下收同音字兔，依反切为透母暮韵。《韵镜》於内转第十二合（原作内转第十二开合，今依杨军《韵镜校笺》一著校正)去声暮韵一等透母位置列兔。则此处所列之兔为《唐韵》之非小韵首字。十五卦韵之辉小韵，傍卦反，下收同音字稗，依反切为母卦韵。《韵镜》於外转第十六合去声卦韵二等母位置列稗，则此处所列之稗为《唐韵》之非小韵首字。  

# 〈三〉《韵镜》所列之字为当删之同音字  

这种情况是指，《韵镜》中已列了《唐韵》中的一个小韵首字，同时又列了一个该小韵首字下面的一个同音字，此同音字从音韵地位上来说当删。这种情况共有四例（4)，占《唐韵》所收小韵首字字数的百分之一点二五（ $1.25\%$ )。如：三烛韵之束小韵，书玉反，下收同音字悚，依反切为书母烛韵。《韵镜》已於内转第二合（原作内转第二开合，今依杨军《韵镜校笺》一著校正）入声烛韵三等书母位置列束，又於二等初母位置列，则此处所列之悚依《唐韵》为当删之同音字。二十九药韵之小韵（或作)，下收同音字，依反切为晓母药韵。《韵镜》已於内转第三十二合入声药韵三等晓母位置列曦，又於群母位置列，若依《唐韵》此处所列之为当删之同音字。  

# 《四〉《韵镜》所列为形误字  

这种情况是指《韵镜》所列的字形有误，有的也有可能是《唐韵》所收字形有误，或还有可能是写法问题。这类情况共有十二例（12)，占《唐韵》所收小韵首字字数的百分之二点七六 $(3.76\%$ )。如：五十三艳韵之小韵，昌艳反，依反切为昌母艳韵。《韵镜》於外转第三十九开去声艳韵三等昌母位置列，当为之列误。三烛韵之楝小韵，丑玉反，依反切为彻母烛韵。《韵镜》於内转第二开（原作内转第二开合，今依杨军《韵镜校笺》一著校正）入声烛韵三等彻母位置列怖，怖当为楝之形误。  

# 《五）《韵镜》所列之字与《唐韵》的反切不合  

这种情况是指，《韵镜》中收有《唐韵》中的小韵首字，但《韵镜》中所列的位置与《唐韵》反切所反映的位置不合。这类情况共有五例（5)，占《唐韵》所收小韵首字字数的百分之一点五七（ $1.57\%$ ）。如：九迄韵之迄小韵，诗乞反，依反切为书母迄韵。《韵镜》於外转第十九开入声迄韵三等晓母位置列迄。若依《唐韵》则位置不合。十月韵之伐小韵，户越反，依反切为匣母月韵。《韵镜》於外转第二十二合入声月韵三等（奉）母位置列伐，若依《唐韵》则位置不合。  

《六）《韵镜》所列之字为当删之重字  

这种情况是指《韵镜》中已列了一个与《唐韵》的小韵首字对等的字，但又在某处列了一个与之完全相同的一个字。这样的字从音韵地位上来说当为当删的重字。这类情况共有五例（5)，占《唐韵》所收小韵首字字数的百分之一点五七$(1.57\%)$ 》。如：五十三艳韵之染小韵，而赡反，依反切为日母艳韵。《韵镜》已於外转第三十九开去声艳韵三等日母位置列染，又於外转第四十开（原作外转第四十合，今依杨军《韵镜校笺》一著校正》去声艳韵四等日母位置列染，则外转第四十开（原作外转第四十合，今依杨军《韵镜校笺》一著校正）所列之染为当删之重字。五十三艳韵之验小韵，鱼反，依反切为疑母艳韵。《韵镜》已於外转第三十九开去声艳韵三等疑母位置列验，又於外转第四十开（原作外转第四十合，今依杨军《韵镜校笺》一著校正）去声艳韵四等疑母位置列验，则外转第四十开（原作外转第四十合，今依杨军《韵镜校笺》一著校正）所列之验为当删之重字。  

《七〉《韵镜》所列之字与《唐韵》所收之小韵首字为同一字的不同形体  

这类情况共有五例（5)，占《唐韵》所收小韵首字字数的百分之一点五七$(1.57\%)$ 。如：三十四啸韵之突小韵，乌叫反，依反切为影母啸韵。《韵镜》於外转第二十五开去声啸韵四等影母位置列。“突”、“窦”当为同一字的不同形体。二十四洽韵之画小韵，女洽反，依反切为泥（娘）母洽韵。《韵镜》於外转第三十九开入声洽韵二等泥（娘）母位置列区。“画”、“区”当为同一字的不同形体。《广韵》即作冈，注云“俗作回”。  

# 〈八）同一音韵地位的字  

这种情况是指，在《广韵》或其他韵书中A小韵首字下收有B同音字，而《唐韵》用B作小韵首字，而B下不见A，《韵镜》所列的也是A。这种情况共有三（3）例，占《唐韵》所收小韵首字字数的百分之零点九四 $\left(0.94\%\right)$ 。如：二十麦韵之小韵，呼麦反，依反切为晓母麦韵。《韵镜》於外转第三十六合入声麦韵二等晓母位置列劃。“劃”、“”当为同一音韵地位的字。《唐韵》、王三作，皆注云“破声”（《唐韵》误作“声破")。《广韵》作割，亦注云“破声”,下收有同音字劃。  

# 〈九〉《韵镜》列有，而《唐韵》未收  

这种情况是指《韵镜》列有该字，但《唐韵》没有收，或《唐韵》该韵没有收（该字可能在其他韵里出现，由于初步统计，不排除这种情况在存在)。这类情况共有四十九例（49)，占《唐韵》所收小韵首字字数的百分之十五点三六$(15.36\%)$ )。如：《韵镜》於内转第十二合（原作内转第十二开合，今依杨军《韵镜校笺》一著校正）去声暮韵一等精母位置列做，《唐韵》十一暮韵未收有此小韵。《韵镜》於外转第十五开去声卦韵二等帮母位置列薛，《唐韵》十五卦韵未收有此小韵。《广韵》、王一、王三皆收有小韵。  

# 〈十〉《唐韵》收有，而《韵镜》未列  

这种情况是指《唐韵》收有该小韵首字，但《韵镜》里相应位置没有列出，在其他地方也没有出现。这类情况共有八例（8)，占《唐韵》所收小韵首字字数的百分之二点五一 $(2.51\%)$ 。如：十六怪韵之小韵，所拜反，依反切为生母怪韵。《韵镜》於外转第十三开去声怪韵二等生母位置未列，於其他位置亦未列。五十五证韵之丞小韵，常证反，依反切为禅母证韵。《韵镜》於内转第四十二开去声证韵三等禅母位置列船母之剩，而未列禅母之丞。

# 第五节 《韵镜》与五代刻本韵书残卷之比较  

# 2.5.1关于五代刻本韵书残卷的基本情况  

这里所说的五代刻本韵书残卷是指“P2014”、“P2015”、“P2016”、“P4747”  

“P5531”。顺序依照周祖谟在《唐五代韵书集存》里的顺序，即“P2016、P4747、P2014 之三、P2015 之三、P2014 之四、P2015 之二、P2014 之六正、P2014 之六反、P2014 之八正、P2014之五、P2014之八反、P2014 之七、P2014 之九、P5531之二、P5531之一、P2014 之二、P5531 之三、P2015 之一、P2014之一”，该顺序也是按此类残卷前后相连的自然顺序而排列的。下面简要介绍一下各残卷的基本情况。  

# 2.5.1.1关于P2016 的基本情况  

五代刻本韵书残叶P2016存有序文七行，切韵平声上全部韵目，前面书有“切韵平声上第一凡二十八韵"；另存有一东韵九行，前六行较为完整，后三行较残。此页背面有七行，据有关学者考证为另一种韵书的一部分，此处不加以整理。据本人初步统计，本残叶仅保存一东韵的一个完整小韵首字（另有一个缺切下字）。  

# 2.5.1.2关于P4747的基本情况  

五代刻本韵书残叶P4747存一叶的上半叶，共计十二行。此残叶所存之字当为一东韵之残，共保存一东韵的六个完整的小韵首字。体例上每小韵的韵目上加有朱点标识，每一纽於第一字下先出训释，后出反切，最后统计此纽所收的同音字字数。又音除用反切外，兼用平上去入四声之间的关系来标注。据周祖谟在《唐五代韵书集存》里的意见，“又伯希和所编四七四七号存东韵十二行的上一半，这正是二0一四（3）东韵一纸所缺的一部分，两者都是印本，.纸叶正相连接，应当粘合在一起，.."。周先生将这两种残叶前后影印在一起，此也将此两种残叶前后相连整理在一起。  

# 2.5.1.3关于P2014之三的基本情况  

五代刻本韵书残叶P2014之三保存平声一东二冬之残，一东保存有八个完整的小韵首字，二冬保存有十一个完整的小韵首字。体例上一东第十个小韵以上韵目上加有朱点，第十个小韵以上至二冬末韵目上不加朱点。每一纽於第一字下先出训释，后出反切，最后统计此纽所收的同音字字数。又音除用反切外，兼用平上去入四声之间的关系来标注。标有板号，一东第三小韵烘下标有“二板”字样。据周祖谟在《唐五代韵书集存》里的意见，“根据照片来看，二〇一六抄写的二十行和二0一四第三叶东韵前面抄写的八行笔迹完全相同，而且文字正相衔接，应当就是一叶，…”。今观P2016和P2014之三，周先生之说是可信的。  

# 2.5.1.4关于P2015之三的基本情况  

五代刻本韵书残叶P2015之三共存一东六行两个完整的小韵首字，二冬全部完整，三钟四行四个完整的小韵首字，总共十九行。体例上，大韵目上加有较大的朱圈，小韵目上加有朱点。每纽於第一字下先出训释，后出反切，最后统计此纽所收的同音字字数。又音除用反切以外，兼用平上去入四声的关系标注。标有板号，三钟第四小韵茸下标有“三板”字样。今按P2015之三前十六行（除去最先两行）与P2014之三后十四行正好相重，除 P2014 之三后半部分在韵目上不加朱点而P2015之三在韵目上加有朱点以外，内容完全相同。且 P2014 标有“二板"，而 P2015 之三标有“三板"。据此我们认为 P2014 之三和 P2015 之三即使不是同一部韵书，也是同一类的韵书，可以前后相连看作一个整体。  

# 2.5.1.5关于P2014之四的基本情况  

五代刻本韵书残叶 P2014之四残存九鱼五个完整的小韵首字,十虞六个完整的小韵首字，共存十九行。体例上，大韵目上加有朱圈，小韵目上加有朱点。每纽於第一字下训释和反切先出后出不一，最后统计此纽所收的同音字字数。又音除用反切外，兼用平上去入四声之间的关系来标注。标有板号“八板”字样。  

2.5.1.6关于P2015之二的基本情况  

五代刻本韵书残叶P2015之二为完整的一板,其中间稍有残缺,共三十四行，为十二齐，残缺前面部分；十三佳，有一小韵韵目难辨；十四皆，小韵首字完整;十五灰，残缺后面部分。体例上，韵目上加有朱点，每纽於第一字下训释和反切先出后出不一，最后统计此纽所收的同音字字数。又音除用反切外，兼用平上去入四声的关系来标注。标有板号“十一板”字样。  

2.5.1.7关于P2014之六的基本情况  

五代刻本韵书残叶P2014之六存正反两面。正面存平声下部分韵目和二十九先及三十仙前面一部分，间有残缺，共计二十五行。背面存三十仙后面部分，间有残缺；三十一宣，后面稍有残缺；三十二萧萧小韵和一个非小韵首字。共计二十五行（前面各有部分残留)。体例上，韵目上加有朱点。每纽於第一字下训释和反切先出后出不一，最后统计此纽所收的同音字字数。又音除用反切外，兼用平上去入四声之间的关系标注，有少数用直音的办法。  

2.5.1.8关于P2014 之八正、P2014之五、P2014之八反的基本情况  

五代刻本韵书残叶P2014之八正面存有十三行；为三十三宵，前后皆有残缺，中间间有残缺；三十四肴，后面有残缺，中间间有残缺。背面存有十三行，为三十四肴，前面有残缺；三十五豪，后面有残缺。五代刻本韵书残叶P2014之五存有九行（前后皆有部分残缺，不计)，为三十四肴，前后皆有残缺。今比观P2014之八正、P2014之五、P2014之八反，P2014之五前面部分与P2014之八正后面部分有相合部分，相合部分内容完全一样；P2014之五后面部分与P2014之八反前面部分也有相合部分，相合部分也完全一样。除去相合部分，P2014之五正好将P2014之八正与P2014之八反连接起来成为一体。据些我们可以说P2014之八正、P2014之五、P2014之八反当为同一种韵书。体例上，韵目上加有朱点。每纽於第一字下训释和反切先出后出不一，最后统计此纽所收的同音字字数。又音除用反切外，兼用平上去入四声之间的关系标注。P2014之五后标有板号“十八板”字样。  

# 2.5.1.9关于P2014之七的基本情况  

五代刻本韵书P2014之七存有十三行，为五十侵，残缺前面部分；五十一盐，残缺后面部分。中间间有残缺。体例上，韵目上加有朱点。每纽於第一字下训释和反切先出后出不一，最后统计此纽所收的同音字字数。又音除用反切外，兼用平上去入四声之间的关系标注。后有板号“二十六板”字样。

# 2.5.1.13关于P2014之二的基本情况

五代刻本韵书共存有四十二个韵，四百三十二个小韵。这里的小韵数指保存完整的，即韵目及其反切，至少是切上字完整。因此实际小韵数比此数目要多。在所存的这些韵及小韵中，保存完整的，指小韵完整，至少是小韵首字及其反切完整，少数反切仅缺切下字也作完整处理，有四个韵，五十个小韵。  

由上面的考察可以看出，这些五代刻本韵书有很多相同相似的地方，正如众多学者认为的那样，以上这些五代刻本韵书残叶应该是同一种韵书。因此，在这里我们将它们放在一起加以整理，统作为五代刻本韵书。但也正如有的学者认为的那样，上述这些韵书残叶的平上部分与入声部分在体例上又有分别，我们在具体处理的时候，应当有所区别和注意。

根据五代刻本韵书的反切，《韵镜》所列之字与之完全相合的情况共有三百零二例（302），占五代刻本韵书所收小韵首字字数的百分之六十九点九一$(69.91\%)$ 。如：P4747一东韵之弓小韵，居隆反，依反切为见母东韵。《韵镜》正於内转第一开平声冬韵三等见母位置列弓。P2014二冬韵之冬小韵，都宗反，依反切为端母冬韵。《韵镜》正於内转第二合（原作内转第二开合，今依杨军《韵镜校笺》一著校正）平声冬韵一等端母位置列冬。

这种情况是指，《韵镜》所列之字不是五代刻本韵书中的小韵首字，而是五代刻本韵书中小韵首字下所收之同音字。这种情况共有三十七例（37)，占五代刻本韵书所列小韵首字字数的百分之八点五六 $(8.56\%)$ 。如：P2014二冬韵之蜂小韵，户冬反，依反切为匣母冬韵，下收有同音字。《韵镜》於内转第二合（原作内转第二开合，今依杨军《韵镜校笺》一著校正）平声冬韵一等匣母位置列。依五代刻本韵书P2014 则此处所列之为非小韵首字。P2014九鱼韵之锄小韵,士鱼反，依反切为崇母鱼韵，下收有同音字。《韵镜》於内转第十一开平声鱼韵二等崇母位置列。依五代刻本韵书 P2014 则此处所列之为非小韵首字。  

# 《三〉《韵镜》所列之字为当删之同音字  

这种情况是指，《韵镜》中已列了五代刻本韵书中的一个小韵首字，同时又列了该小韵首字下面的一个同音字，此同音字从音韵地位上来说当删。这种情况共有一例（1)，占五代刻本韵书所收小韵首字字数的百分之零点零三（ $.0.03\%)$ 。如：P5531十三骇韵之楷小韵，口骇反，依反切为溪母骇韵，下收有同音字锴。《韵镜》已於外转第十三开上声骇韵二等溪母位置列楷，又於见母位置列锴。依五代刻本韵书P5531则此处所列之错为当删之同音字。  

# 《四〉《韵镜》所列为形误字  

这种情况是指《韵镜》所列的字形有误，有很大一部分是五代刻本韵书所收字形有误，或还有可能是写法问题。这类情况共有十一例（11)，占五代刻本韵书所收小韵首字字数的百分之二点五五 $(2.55\%)$ 。如：P2014二冬韵之小韵，此浣反，依反切为清母冬韵。《韵镜》於内转第二合（原作内转第二开合，今依杨军《韵镜校笺》一著校正）平声冬韵一等清母位置列。此处所列之当为之或体之形误。P2015十二齐韵之雞小韵，古兮反，依反切为见母齐韵。《韵镜》於外转第十三开平声齐韵四等见母位置列。此处所列之当为之形误。

# 第六节 《韵镜》与《广韵》之比较  

# 2.6.1关于《广韵》的基本情况  

《广韵》一书共有二百零六韵。其中上平声二十八韵，下平声二十九韵，上声五十五韵，去声六十韵，入声三十四韵。  

《广韵》一书共收有三千八百七十四个小韵首字。其中平声有一千一百五十六个（上平声五百七十四个，下平声五百八十二个)，上声有九百三十五个，去声有一千零九十一个，入声有六百九十二个。（其中有十四个小韵首字的反切在注释文字之前。其中平声有两个，上声有六个，去声有六个。)  

《广韵》一书正文分为五卷（上平、下平、上声、去声、入声)，二百零六韵。每一卷前面标明卷名和卷次，然后列出所包含的各韵韵目，注出其反切韵次，同用独用的情况，最后按韵次排列出各韵的情况。每一个小韵上方用“○”标识，每一个小韵下先解释，后列反切，反切用“xx切”，再标出所收同音字的字数。有少数小韵反切在注释文字之前，当是后人所增。详细情况参见本人的统计数字。  

# 2.6.2《韵镜》与《广韵》之比较  

经过比较、分析和统计，《韵镜》所列之字与《广韵》所收小韵首字之关系主要有以下几种情况：  

# （一〉完全相合的情况  

根据《广韵》的反切，《韵镜》所列之字与之完全相合的情况共有三千二百二十七例（3227），占《韵镜》所收字数的百分之八十二点八九 $(82.89\%)$ ，占《广韵》所收小韵首字的字数百分之八十三点三零： $\langle83.30\%\rangle$ 。如：一东韵之同小韵，徒红切，依反切为定母东韵。《韵镜》正於内转第一开平声东韵一等定母位置列同。二冬韵之攻小韵，古冬切，依反切为见母冬韵。《韵镜》正於内转第二合（原作内转第二开合，今依杨军《韵镜校笺》一著校正）平声冬韵一等见母位置列攻。《二》《韵镜》所列之字为《广韵》之非小韵首字．  

这种情况是指，《韵镜》所列之字不是《广韵》中的小韵首字，而是《广韵》中小韵首字下所收之同音字。这种情况共有二百五十一例（251)，占《韵镜》所收字数的百分之六点四五 $(6.45\%)$ ，占《广韵》所收小韵首字字数的百分之六点四八( $6.48\%$ )。如：三钟韵之小韵，息恭切，依反切为心母钟韵，下收同音字淞。《韵镜》於内转第二合（原作内转第二开合，今依杨军《韵镜校笺》一著校正）平声钟韵四等心母位置列淞。依《广韵》此处所列之淞为非小韵首字。八微韵之斐小韵，甫微切，依反切为帮（非）母微韵，下收同音字非。《韵镜》於内转第十合平声微韵三等帮（非）母位置列非。依《广韵》此处所列之非为非小韵首字。  

# 《三）《韵镜》所列之字为当删之同音字  

这种情况是指，《韵镜》中已列了《广韵》中的一个小韵首字，同时又列了一个该小韵首字下面的一个同音字，此同音字从音韵地位上来说当删。这种情况共有七十九例（79)，占《韵镜》所列字数的百分之二点零三 $(2.03\%)$ ，占《广韵》所收小韵首字字数的百分之二点零四 $(2.04\%)$ 。如：一东韵之融小韵，以戎切，依反切为喻母四等东韵，下收同音字彤。《韵镜》已於内转第一开平声东韵四等喻母位置列融，又於喻母三等位置列彤。依《广韵》此处所列之彤为当删之同音字。八微韵之祈小韵，渠希切，依反切为群母微韵，下收同音字顾。《韵镜》已於内转第九开平声微韵三等群母位置列祈，又於内转第十合平声微韵三等群母位置列顾。依《广韵》则内转第十合所列之顾为当删的同音字。  

# 《四）《韵镜》所列为形误字  

这种情况是指《韵镜》所列的字形有误，有的也有可能是《广韵》所收字形有误，或还有可能是写法问题。这类情况共有一百三十七例（137)，占《韵镜》所列字数的百分之三点五二 $(3.52\%)$ ，占《广韵》所收小韵首字字数的百分之三点五四 $(3.54\%)$ 。如：十二齐韵之離小韵，古奚切，依反切为见母齐韵。《韵镜》於外转第十三开平声齐韵四等见母位置列。此处所列之当为之形误。八戈韵之赢小韵，落戈切，依反切为来母戈韵。《韵镜》於内转第二十八合平声戈韵一等来母位置列赢，赢当为赢之形误。  

《五〉《韵镜》所列之字与《广韵》的反切不合  

这种情况是指，《韵镜》中收有《广韵》中的小韵首字，但《韵镜》中所列的位置与《广韵》反切所反映的位置不合。这类情况共有五十二例（52)，占《韵镜》所列字数的百分之一点三四！ $\langle1.34\%\rangle$ ，占《广韵》所收小韵首字字数的百分之一点三二 $(1.32\%)$ 。如：十二庚韵之荣小韵，永兵切，依反切为匣（喻三）母庚韵。《韵镜》於外转第三十四合平声庚韵喻母四等位置列荣，位置不合。十四清韵之营小韵，余顷切，依反切为喻母四等清韵。《韵镜》於外转第三十四合平声清韵喻母三等位置列营，位置不合。  

# 《六）《韵镜》所列之字为当删之重字  

这种情况是指《韵镜》中已列了一个与《广韵》的小韵首字对等的字，但又在某处列了一个与之完全相同的一个字。这样的字从音韵地位上来说当为当删的重字。这类情况共有三十八例（38)，占《韵镜》所列字数的百分之零点九八$\left(0.98\%\right)$ ，占《广韵》所收小韵首字字数的百分之零点九八！ $\langle0.98\%\rangle$ 。如：十四清韵之轻小韵，去盈切，依反切为溪母清韵。《韵镜》已於外转第三十三开平声清韵四等清母位置列轻，又於外转第三十五开平声清（青）韵四等清母位置列轻，则外转第三十五开所列之轻为当删之重字。四十静韵之颈小韵，居郢切，依反切为见母静韵。《韵镜》已於外转第三十三开上声静韵四等见母位置列颈，又於外转第三十五开上声静韵三等见母位置列颈，则外转第三十五开所列之颈为当删之重字。  

《七〉《韵镜》所列之字字为《广韵》非小韵首字且位置与《广韵》的反切不合  

这类情况共有四例（4)，占《韵镜》所列字数的百分之零点一零 $(0.10\%)$ 占《广韵》所收小韵首字字数的百分之零点一零 $(0.10\%)$ 。如：十四祭韵之桑小韵，楚切，依反切为初母祭韵，下收有同音字。《韵镜》於外转第十四合去声祭韵三等昌母位置列囊。依《广韵》则此处所列之囊为非小韵首字且所列位置与《广韵》的反切不合。十四祭韵之祭小韵，子例切，依反切为精母祭韵，下收有同音字德。《韵镜》於内转第十六合去声泰韵一等喻为母位置列德，依《广韵》则此处所列之为非小韵首字且所列位置与《广韵》的反切不合。  

# 〈八〉《韵镜》所列之字字形有误且为当删之同音字  

这种情况是第三种情况和第四种情况的综合。这类情况共有五例(5)，占《韵镜》所列字数的百分之零点一三 $(0.13\%)$ ，占《广韵》所收小韵首字字数的百分之零点一三 $(0.13\%)$ 。如：十九隐韵之谨小韵，居隐切，依反切为见母隐韵，下收有同音字，周祖谟校为。《韵镜》已於外转第十九开上声隐韵见母位置列谨，又於外转第十七开上声韵三等见母位置列。依《广韵》则此处所列之字形有误且为当删之同音字。二沃韵之瑁小韵，莫沃切，依反切为明母沃韵，下收有同音字媚。《韵镜》已於内转第二合（原作内转第二开合，今依杨军《韵镜校笺》一著校正）入声沃韵一等明母位置列瑁，又於内转第二合（原作内转第二开合，今依杨军《韵镜校笺》一著校正）入声烛韵三等明（微）母位置列。当为媚之形误。依《广韵》则此处所列之婿为字形有误且为当删之同音字。  

《九）《韵镜》所列之字与《广韵》之反切经周校后合  

这种情况是指，《广韵》中的反切经周祖谟校正后正好与《韵镜》所列之字相合。这种情况共有十三例（13)，占《韵镜》所列字数的百分之零点三三 $(0.33\%)$ 占《广韵》所收小韵首字字数的百分之零点三四 $\left(0.34\%\right)$ 。如：十六韵之小韵，士忍切，依反切为崇母韵，周祖谟《广韵校本》校作七忍切，则为清母韵。《韵镜》於外转第十七开上声韵四等清母位置列，与周祖谟校过之反切合。二十八称韵之浅小韵，士演切，依反切为崇母称韵。周祖谟《广韵校本》校作七演切，则为清母称韵。《韵镜》於外转第二十一开上声称韵四等清母位置列浅，与周祖谟校过之反切合。  

《十）《韵镜》所列之字与《广韵》所收之小韵首字为同一字的不同形体  

这类情况共有七例（7)，占《韵镜》所列字数的百分之零点一八 $(0.18\%)$ ?占《广韵》所收小韵首字字数的百分之零点一八 $(0.18\%)$ 。如：十一模韵之蘇小韵，素姑切，依反切为心母模韵。《韵镜》於内转第十二合（原作内转第十二开合，今依杨军《韵镜校笺》一著校正）平声模韵一等心母位置列。“蘇”、“”当为同一字之不同形体。二十一混韵之小韵，苏本切，依反切为滂母混韵。《韵镜》於外转第十八合上声混韵一等滂母位置列。“”、“蝌”当为同一字的不同形体。  

〈十一〉《韵镜》所列之字字形有误且为非小韵首字  

这种情况是第二种情况和第四种情况的综合。这类情况共有六例(6)，占《韵镜》所列字数的百分之零点一五 $(0.15\%)$ ，占《广韵》所收小韵首字字数的百分之零点一五 $(0.15\%)$ 。如：二十四盐韵之小韵，处占切，依反切为昌母盐韵，下收同音字檐。《韵镜》於外转第三十九开平声盐韵三等昌母位置列。依《广韵》此处所列之当为之形误且为非小韵首字。三十九过韵之嬴小韵，鲁过切,依反切为来母过韵，下收同音字。《韵镜》於内转第二十八合去声过韵一等来母位置列。依《广韵》此处所列之当为之形误且为非小韵首字。  

《十二》《韵镜》所列之字字形有误且与《广韵》的反切不合  

这种情况是第四种情况和第五种情况的综合。这种情况共有三例（3)，占《韵镜》所列字数的百分之零点零八 $(0.08\%)$ ，占《广韵》所收小韵首字字数的百分之零点零八（ $.0.08\%)$ 。如：十三耕韵之小韵，女耕切，依反切为泥（娘）母耕韵。《韵镜》已於外转第三十五开平声耕韵二等泥（娘）母位置列，偏当为之形误，又於外转第三十五开平声庚韵二等泥（娘）母位置列，当为之形误。则外转第三十三开所列之偏是字形有误且与《广韵》的反切不合（同时也是当删之重字)。十五韵之小韵，丁刮切，依反切为端母韵。《韵镜》於外转第二十二合入声二等知母位置列。当为之形误，且与《广韵》的反切不合（此为以端切知，类隔切，处理为不合）。  

# 《十三）《韵镜》所列之字字形有误且为当删之重字  

这种情况是第四种情况和第六种情况的综合。这类情况共有两例(2)，占《韵镜》所收字数的百分之零点零五 $(0.05\%)$ ，占《广韵》所收小韵首字字数的百分之零点零五 $(0.05\%)$ 。如：四十三映韵（《韵镜》作敬，敬字缺末笔）之小韵,乌横切，依反切为影母映韵。《韵镜》於外转第三十四合去声敬韵二等影母位置列，又於外转第三十六合去声韵二等影母位置列。依《广韵》则外转第三十六合去声韵所列之为字形有误且为当删之重字。十三耕韵之小韵，女耕切，依反切为泥（娘）母耕韵。《韵镜》：已於外转第三十五开平声耕韵二等泥（娘)母位置列，偏当为之形误，又於外转第三十五开平声庚韵二等泥（娘）母位置列，当为之形误。则外转第三十三开所列之为字形有误且为当删之重字。  

《十四》《韵镜》所列当为据《广韵》又音所列  

此类情况共有一例（1)，占《韵镜》所列字数的百分之零点零三 $(0.03\%)$ 占《广韵》所收小韵首字字数的百分之零点零三 $(0.03\%)$ 。如：三十五马韵之姐小韵，兹野切，依反切为精母马韵，下收有同音字担，又才也切。《韵镜》於外转第二十九开（原作内转第二十九开，今依杨军《韵镜校笺》一著校正）上声马韵四等精母位置列姐，又於从母位置列担。依《广韵》则此处所列之担可视为据又音所列。  

# 〈十五〉《韵镜》列有，而《广韵》未收  

这种情况是指《韵镜》列有该字，但《广韵》没有收，或《广韵》该韵没有收(该字可能在其他韵里出现，由于初步统计，不排除这种情况存在)。这类情况共有六十九例（69)，占《韵镜》所列字数的百分之一点七七 $(1.77\%)$ ，占《广韵》所收小韵首字字数的百分之一点七八（ $.1.78\%$ ）。如：《韵镜》於外转第十八合平声淳韵三等书母位置列蝽，《广韵》十八淳韵未收此小韵。《韵镜》於外转第三十五开平声耕韵二等彻母位置列撑，《广韵》十三耕韵未收此小韵。.  

《十六〉《广韵》收有，而《韵镜》未列  

这种情况是指《广韵》收有该小韵首字，但《韵镜》里相应位置没有列出，在其他地方也没有出现。这类情况共有一百七十四例（174)，占《韵镜》所列字数的百分之四点四七 $(4.47\%)$ ，占《广韵》所收小韵首字字数的百分之四点四九$(4.49\%$ )。如：《广韵》十四清韵之贞小韵（贞字缺末笔)，陟盈切，依反切为知母清韵。《韵镜》於外转第三十三开清韵三等知母位置未列。《广韵》十六蒸韵之殊小韵，山矜切，依反切为生母蒸韵。《韵镜》於内转第四十二开平声蒸韵二等生母位置未列。

# 第三章 结论  

作为汉语语音史尤其是中古汉语研究重要资料的《韵镜》，一直得到众多学者的重视，无数前贤也对此做了不少研究工作。但由于《韵镜》一书的诸多特殊性，其作者至今仍不可考，因而其成书年代至今仍旧众说纷坛，其列字所据到底为何种韵书至今也没有达成一致的意见。导致这些问题的本质在于目前对《韵镜》的研究还不够，对《韵镜》的研究尤其是对《韵镜》与现存的相关韵书之间关系的研究还不够全面。  

《韵镜》作为一种韵图，当是据韵书，主要是韵书中的小韵首字而作，这一点在学术界是没有异议的。因此比较考察《韵镜》所列之字与相关各韵书所收之字主要是小韵首字之间的异同及相关情况对判断《韵镜》一书的成书年代及其所据为何种韵书是很有价值的。因此本文就着眼于此，将《韵镜》所列之字与各相关韵书作了一个全面的比较，以期找到解决问题的途径。同时，《韵镜》一书和相关各韵书的韵目在排列的前后顺序上也显示出一定的特点，这对本文的研究也是很有意义的，在此一并加以分析考察。  

综合本文的考察和分析，现得出以下的结论：  

# 一、《韵镜》所依据的韵书  

从《韵镜》和相关各韵书中“覃、谈”“蒸、登”韵目排列的先后次序上可以看出，《韵镜》当是据前一阶段的韵书（包括S2071、P2011、“王三"、《唐韵》)而作的，当然不能排除依据五代刻本韵书而作的可能性。从《韵镜》与相关各韵书四声相承的关系来看，《韵镜》当是据后一阶段的韵书（包括李舟《切韵》、《广韵》）而作的，同时可以排除依据五代刻本韵书而作的可能性。从《韵镜》与相关各韵书收字主要是小韵首字的比较上可以看出，《韵镜》与《唐韵》的相合百分比是最高的，《韵镜》与“五代刻本韵书”的相合百分比是最低的，只有百分之六十左右。这说明了在相关各韵书中，《韵镜》一书与《唐韵》最为接近，《韵镜》当是依据《唐韵》同时期或前后的某一种韵书来列字的。从上述相关各韵书收有而《韵镜》未列的情况来看，在《韵镜》列字与相关韵书所收小韵之比较（表一）中，《唐韵》的百分率是最低的；在《韵镜》列字与相关韵书所收小韵之比较（表二）中，《唐韵》的百分率仪次於S2071的百分率。这也从一个方面说明了《韵镜》与《唐韵》的关系最为接近，《韵镜》据《唐韵》同时期或前后的某一种韵书来列字的可能性是最大的。从上述《韵镜》所列之字为相关各韵书中的非小韵首字的情况来看，《唐韵》的百分率也是最低的。这又从一个方面说明了《韵镜》与《唐韵》的关系最为接近，《韵镜》据《唐韵》同时期或前后的某一种韵书来列字的可能性是最大的。综合考虑以上诸种情况，我们得出本文一致的结论，即《韵镜》当是依据《唐韵》与李舟《切韵》之间的某种韵书来列字的。这种韵书体现了韵书发展过程中的一种过渡和前后相承的关系。它既具有前一阶段韵书的特点，即“蒸、登”的位置比较靠后，“阳、唐”的位置比较靠前，又具有后一阶段韵书的特点，即四声相承，次序井然，同时又符合《韵镜》与《唐韵》相合百分率是最高的等诸多事实。这种韵书虽然没有在现存的韵书中出现，但这种韵书的存在是极为可能的。这也与平山久雄在《韵镜二事》一文中所作的假设是相一致的，即历史上确实存在过这样的一种韵书。  

# 二、《韵镜》的成书年代  

上面我们已经大致确定《韵镜》是依据《唐韵》与李舟《切韵》之间的某种韵书来列字的。那么《韵镜》一书的成书年代最早可能在《唐韵》成书年代与李舟《切韵》成书年代之间，即最早可能在公元762年（《唐韵》成书年代，此据周祖谟《唐五代韵书集存》中的观点，“据此推测此书可能写於代宗之世"）至公无785年（李舟《切韵》成书年代，此据王国维在《李舟切韵考》中的观点：“李舟作《切韵》当在唐代宗、德宗之世”）之间。  

本文通过对《韵镜》与相关韵书之间的比较研究，对《韵镜》列字所依据的韵书及《韵镜》一书的成书年代作出了初步的判断。但此一判断能否解决《韵镜》一书列字所依据的韵书和《韵镜》一书的成书年代这一长期没有定论的问题，还需要得到进一步的验证和考验，本人也期待在以后的工作中能够找到其他方面的材料和证据，从而将对《韵镜》的研究进一步深入下去。
